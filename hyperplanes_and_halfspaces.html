§0where we are

§2.1 built the vocabulary — affine, convex, conic; hulls; the segment test. §2.2 starts the catalogue of concrete convex sets, and the first two are the most-used of all: the hyperplane (affine) and the halfspace (convex). This is also where the halfspace — the canonical "prove convex via the segment test" example — finally arrives, so you'll run that proof here, the one you reached for back in §2.1.4.

The warm-up zoo (§2.2 preamble)

A quick §2.1 consolidation before the two stars. Each simple set, sorted by what it is:

  • affine
    ∅, a single point {x₀}, and all of Rⁿ are affine (hence convex). A line is affine — and a subspace (so a convex cone) when it passes through 0.
  • convex
    A segment is convex but not affine (unless it's a point). A ray {x₀+θv | θ≥0}, v≠0, is convex, not affine.
  • cone?
    That ray is a convex cone iff x₀ = 0. B&V states the "if"; the "only if" is closure under scaling — merely containing 0 isn't enough (e.g. x₀=−v puts 0 on the ray, yet it's not a cone).
  • both
    A subspace is affine and a convex cone.
§1the hyperplane — affine
A hyperplane is a set of the form \(\{x \mid a^{\mathsf T}x = b\}\), where \(a \in \mathbf{R}^n,\ a \neq 0,\ b \in \mathbf{R}\).B&V §2.2.1 · L1419

It's the solution set of one linear equation, so it's affine (§2.1.2). Three ways to read the same object:

  • algebra
    The points whose inner product with a is the constant b — a level set of the linear functional aᵀx.
  • geometry
    A flat with normal a: the direction a is perpendicular to the whole surface.
  • offset
    b sets the offset — the signed distance from the origin is b/‖a‖.

Pick any point x₀ on the hyperplane (so aᵀx₀ = b) and recentre:

\[ \{x \mid a^{\mathsf T}x = b\} \;=\; \{x \mid a^{\mathsf T}(x-x_0)=0\} \;=\; x_0 + a^{\perp}, \qquad a^{\perp}=\{v \mid a^{\mathsf T}v = 0\}. \]

So a hyperplane is an offset x₀ plus every vector orthogonal to the normal a. Since a⊥ = N(aᵀ) has dimension n−1 (rank–nullity on the rank-1 functional aᵀ), a hyperplane is an affine set of dimension n−1 — codimension 1.

§2the halfspace — convex
A (closed) halfspace is a set of the form \(\{x \mid a^{\mathsf T}x \le b\}\), \(a \neq 0\) — the solution set of one linear inequality. Halfspaces are convex, but not affine.B&V §2.2.1 · L1442

Equation → hyperplane (affine). Inequality → halfspace (convex, not affine). The "≤" is the whole difference.

The design crux — prove convex by the segment test

Take x₁, x₂ ∈ H (so aᵀx₁ ≤ b and aᵀx₂ ≤ b) and any θ ∈ [0,1]. We must show the convex combination stays in H.

Claim: \(H=\{x\mid a^{\mathsf T}x\le b\}\) is convexprove via segment test
\[ a^{\mathsf T}\!\big(\theta x_1 + (1-\theta)x_2\big) = \theta\,(a^{\mathsf T}x_1) + (1-\theta)\,(a^{\mathsf T}x_2) \]
linearity of the map \(x\mapsto a^{\mathsf T}x\)
\[ \le \theta\, b + (1-\theta)\, b \]
the crux: \(\theta\ge 0\) and \(1-\theta\ge 0\) let us scale \(a^{\mathsf T}x_i\le b\) without flipping the "≤"
\[ = b. \qquad \therefore\ \theta x_1 + (1-\theta)x_2 \in H. \quad\blacksquare \]
the weights sum to 1, so the bound collapses back to \(b\)
!

The nonnegativity of the weights is load-bearing. Multiply aᵀx₁ ≤ b by a negative θ and it flips to ≥, breaking the bound. That is exactly why a halfspace is convex (θ∈[0,1], weights stay ≥0) but not affine — affine demands all θ∈R, including negatives, which escape the set.

§3which side · the angle picture

a points out of the "≤" side: {aᵀx ≥ b} extends along a, while {aᵀx ≤ b} extends along −a. So a is the outward normal of the halfspace. Which side is a point on? Read the sign of aᵀx − b:

  • ≤ 0
    inside {aᵀx ≤ b}
  • > 0
    the other side
  • = 0
    on the boundary hyperplane

Recentred at a boundary point x₀, the test becomes aᵀ(x−x₀) ≤ 0. Because aᵀ(x−x₀) = ‖a‖‖x−x₀‖cos(∠), the sign is the angle:

∠

The "≤" halfspace = x₀ + every vector making an obtuse or right angle (≥ 90°) with the outward normal a. (Fig 2.8: x₁−x₀ acute with a → not in; x₂−x₀ obtuse → in.)

Boundary & the open halfspace

The boundary of {aᵀx ≤ b} is exactly the hyperplane {aᵀx = b}. Its interior is the open halfspace {aᵀx < b} — the strict inequality.

§4design moves — what mastery looks like

M1Prove a halfspace convex via the segment test

Run the three lines above and name where θ≥0 is used. The grader wants to see you flag the nonnegativity step — that's the load-bearing move, not the algebra around it.

M2Build a halfspace from geometry

Given a normal a and a point x₀, set b = aᵀx₀. Then {aᵀx=b} is the hyperplane through x₀, and the "≤"-halfspace sits on the −a side.

M3Know what scaling does

(a,b) → (αa, αb) with α > 0 gives the same sets. With α < 0 the inequality flips — you get the other halfspace.

Main points for quick review

Hyperplane

\(\{a^{\mathsf T}x=b\}\), \(a\neq0\): affine, normal \(a\), offset \(b\), equals \(x_0+a^{\perp}\) (dim \(n{-}1\), codim 1). Equation → affine.

Halfspace

\(\{a^{\mathsf T}x\le b\}\), \(a\neq0\): convex by the segment test, but not affine. Inequality → convex.

The convexity proof

\(a^{\mathsf T}(\theta x_1+(1{-}\theta)x_2)\le \theta b+(1{-}\theta)b=b\). The \(\theta\ge0\) step — nonnegative weights preserve "≤" — is the crux.

Which side

\(a\) = outward normal of \(\{a^{\mathsf T}x\le b\}\). Sign of \(a^{\mathsf T}x-b\) picks the side; \(a^{\mathsf T}(x-x_0)\le0\) means an obtuse angle with \(a\).

Boundary & build

Boundary = the hyperplane; open halfspace = strict \(<\). Build from geometry: \(b=a^{\mathsf T}x_0\); scaling by \(\alpha>0\) preserves the sets.
