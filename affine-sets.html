Affine sets

A set is affine when it is closed under the whole line through any two of its points — not just the segment between them.

Definition

L1173–1176

A set \(C \subseteq \mathbb{R}^n\) is affine if the line through any two distinct points of \(C\) lies in \(C\):

\[ x_1, x_2 \in C,\ \ \theta \in \mathbb{R} \quad\Longrightarrow\quad \theta x_1 + (1-\theta)x_2 \in C. \]

Two load-bearing facts hide in that line. First, \(\theta\) ranges over all of \(\mathbb{R}\) — the whole line, not the segment. Second, the coefficients \(\theta\) and \(1-\theta\) sum to 1. Hold onto both; the rest of the section is consequences of them.

Forward pointer · not used here

Keep only the segment — restrict \(\theta \in [0,1]\) — and you get convex sets (§2.1.4). The hero figure above is exactly this distinction made draggable.

Affine combinations

L1177–1203

An affine combination is any weighted sum whose weights sum to one — signs unrestricted:

\[ \theta_1 x_1 + \cdots + \theta_k x_k, \qquad \theta_1 + \cdots + \theta_k = 1. \]

By induction, an affine set contains every affine combination of its points, not just two-point ones.

Caveat for the induction. To nest a \(k\)-point combination into two-point combinations, peel off a point whose coefficient \(\neq 1\), so the remaining weights have nonzero sum and can be rescaled back to one.
3

Subspace + offset — the structural heart

L1204–1216

If \(C\) is affine and \(x_0 \in C\), then \(V = C - x_0\) is a subspace. This is the result everything else leans on, so the proof is worth holding — the trick recurs.

Proof — pad the coefficients to sum to 1

For \(v_1, v_2 \in V\) and scalars \(\alpha, \beta\):

\[ \alpha v_1 + \beta v_2 + x_0 = \alpha(v_1+x_0) + \beta(v_2+x_0) + (1-\alpha-\beta)x_0 \ \in C \]

because the three points lie in \(C\) and \(\alpha + \beta + (1-\alpha-\beta) = 1\) — an affine combination. Hence \(\alpha v_1 + \beta v_2 \in V\). The move: to use affineness, pad the coefficients so they sum to one.

Consequences cascade: \(C = V + x_0\) (a subspace plus an offset); \(V\) is independent of which \(x_0 \in C\) you pick; and we define \(\dim C := \dim V\).

Precision point

An affine set is a translated subspace, so it is not generally unbounded. If \(V = \{0\}\), then \(C\) is a single point — bounded, dimension 0. "Affine" means flat, not big.

Example 2.1 — the solution set of Ax = b

L1218–1227

The solution set \(C = \{x : Ax = b\}\) is affine, because for \(x_1, x_2 \in C\):

\[ A\big(\theta x_1 + (1-\theta)x_2\big) = \theta b + (1-\theta)b = b. \]

subspaceThe associated subspace is \(\mathcal{N}(A)\), the nullspace of \(A\).
dimension\(\dim C = n - \operatorname{rank} A\), by rank–nullity.
converseEvery affine set is the solution set of some linear system. So affine \(\iff Ax = b\), both directions.
Don't conflate — the §2.1.2 diagnostic trap

Passing through the origin (being a subspace) and being feasible are different conditions.

Subspace \(\iff b = 0\): since \(0 \in C \iff A\cdot 0 = b \iff b = 0\).
Feasible \(\iff b \in \operatorname{range}(A)\).

So \(b = 0 \Rightarrow\) subspace; \(b \neq 0\) but feasible \(\Rightarrow\) affine yet not a subspace.

Tie-back to §1.2.1. When \(A^{\top}A\) is singular, the least-squares minimizers \(\{x : A^{\top}A\,x = A^{\top}b\}\) form an affine set — a translate of \(\mathcal{N}(A)\) — not a single point.
5

Affine hull

L1231–1237

The affine hull \(\operatorname{aff} C\) is the set of all affine combinations of points of \(C\) — equivalently, the smallest affine set containing \(C\):

\[ \operatorname{aff} C = \bigcap\,\{\, S : S \text{ affine},\ C \subseteq S \,\}. \]

If \(S\) is affine and \(C \subseteq S\), then \(\operatorname{aff} C \subseteq S\). Two distinct points hull to a line; three non-collinear points hull to a plane.

Expert lens

Four things a diagnostician keeps in the front pocket:

Quick review

Main points

  • Affine set: closed under the whole line \(\theta x_1 + (1-\theta)x_2,\ \theta \in \mathbb{R}\) (coeffs sum to 1) — not just the segment.
  • Affine combination: \(\sum \theta_i x_i\) with \(\sum \theta_i = 1\) (any sign); affine sets contain all of them.
  • Structure: \(C = V + x_0\), where \(V = C - x_0\) is a subspace (pad with \((1-\alpha-\beta)\) so coeffs sum to 1); \(V\) independent of \(x_0\); \(\dim C := \dim V\). A point is affine (dim 0) — affine ≠ unbounded.
  • Example 2.1: \(\{x : Ax = b\}\) is affine; \(V = \mathcal{N}(A)\), \(\dim = n - \operatorname{rank} A\); converse: every affine set is some \(Ax = b\).
  • Subspace \(\iff b = 0\) (through origin); feasible \(\iff b \in \operatorname{range}(A)\) — different conditions, don't conflate.
  • Affine hull: all affine combinations = smallest affine set containing \(C\).
