Minimum & minimal, seen
through dual inequalities

Characterize the extreme elements of a set by minimizing linear functionals λᵀz. For the minimum the test is clean and iff; for the minimal a real gap opens between sufficient and necessary — and convexity is what closes part of it.

§0where we are — the dual view of §2.4.2

§2.4.2 defined the minimum and minimal elements of a set S with respect to a generalized inequality ≼K. §2.6.3 characterizes them dually — by minimizing linear functionals λᵀz over S. The results are clean for the minimum, but the minimal case has a genuine gap between the sufficient and necessary conditions, and convexity is exactly what closes (part of) it. This is precisely scalarization in multi-objective optimization.

§1dual characterization of the minimum element
\(x\) is the minimum element of \(S\) (w.r.t. \(\preceq_K\)) \(\iff\) for all \(\lambda\succ_{K^*}0\), \(x\) is the unique minimizer of \(\lambda^{\mathsf T}z\) over \(z\in S\). B&V §2.6.3 · L2800–2841

Geometrically: for any λ ≻K* 0, the hyperplane {z | λᵀ(z − x) = 0} is a strict supporting hyperplane to S at x — it touches S only at x, with all of S on one side. Convexity of S is not required (Fig 2.23).

\[ z\neq x:\quad z-x\succeq_K 0,\ \neq 0,\ \ \lambda\succ_{K^*}0 \ \Rightarrow\ \lambda^{\mathsf T}(z-x)>0 \]
strict characterization (§2.6.2): a strictly-dual-positive \(\lambda\) paired against a nonzero \(K\)-nonnegative vector is \(>0\)
\[ \lambda^{\mathsf T}z>\lambda^{\mathsf T}x \ \ \forall z\neq x \ \Rightarrow\ x \text{ is the unique minimizer} \]

⟸  contrapositive — not the minimum ⟹ some λ is beatena nearby dual λ
\[ x \text{ not the minimum} \ \Rightarrow\ \exists z\in S:\ z\not\succeq_K x,\ \text{i.e. } z-x\not\succeq_K 0 \]

\[ \exists\,\tilde\lambda\succeq_{K^*}0:\ \tilde\lambda^{\mathsf T}(z-x)<0 \ \Rightarrow\ \lambda^{\mathsf T}(z-x)<0 \ \ \forall \lambda\succ_{K^*}0 \text{ near }\tilde\lambda \]

Ex Athe minimum characterization on a concrete set

Take S = {(1,1), (1,3), (3,1), (2,2)} in R² with the componentwise order (K = R²₊, self-dual so K* = R²₊, and λ ≻K* 0 means λ = (λ₁,λ₂) with both positive). Recall from §2.4.2 that (1,1) is the minimum of S. For every λ ≻ 0, minimizing λᵀz picks out (1,1) uniquely — the hyperplane through it touches S nowhere else.

§2dual characterization of minimal elements — the gap

For minimal elements the sufficient and necessary conditions do not match. This gap is the whole content of the section.

Sufficient — always holds, no convexity

If \(\lambda\succ_{K^*}0\) and \(x\) minimizes \(\lambda^{\mathsf T}z\) over \(S\), then \(x\) is minimal.B&V §2.6.3 · L2843–2917 · Fig 2.24
a strict-λ minimizer is minimalno convexity used
\[ \text{suppose not minimal:}\ \exists z\in S,\ z\neq x,\ z\preceq_K x \ \Rightarrow\ x-z\succeq_K 0,\ \neq 0 \]
something in \(S\) sits strictly below \(x\)
\[ \lambda\succ_{K^*}0 \ \Rightarrow\ \lambda^{\mathsf T}(x-z)>0 \ \Rightarrow\ \lambda^{\mathsf T}z<\lambda^{\mathsf T}x \]
strict pairing again — but this contradicts \(x\) minimizing \(\lambda^{\mathsf T}z\) over \(S\) ∎
!

The converse fails in general. A point can be minimal yet not a minimizer of λᵀz for any λ (Fig 2.25). Convexity is essential for any converse.

Necessary — needs convexity, and only nonzero λ

If \(S\) is convex and \(x\) is minimal, then there exists a nonzero \(\lambda\succeq_{K^*}0\) with \(x\) a minimizer of \(\lambda^{\mathsf T}z\) over \(S\).B&V §2.6.3 · L2843–2917 · Fig 2.26
minimal + convex ⟹ a separating dual λseparate two convex sets
\[ x \text{ minimal} \ \iff\ \big((x-K)\setminus\{x\}\big)\cap S=\varnothing \]
nothing in \(S\) lies strictly below \(x\) in the \(K\)-order
\[ \exists\,\lambda\neq0,\ \mu:\quad \lambda^{\mathsf T}(x-y)\le\mu\ \ \forall y\in K,\qquad \lambda^{\mathsf T}z\ge\mu\ \ \forall z\in S \]
separate the convex sets \((x-K)\setminus\{x\}\) and \(S\); bounded on the cone forces \(\lambda\succeq_{K^*}0\)
\[ x\in S \text{ and } x\in x-K \ \Rightarrow\ \lambda^{\mathsf T}x=\mu \ \Rightarrow\ x \text{ attains } \min_{z\in S}\lambda^{\mathsf T}z \]
\(x\) sits in both sets, so it hits the separating value ∎
!

Two things you cannot do. (1) You cannot strengthen the necessary λ to λ ≻K* 0: a minimal point of a convex S may fail to minimize λᵀz for any strictly-positive λ — only a boundary λ ≽K* 0 works (Fig 2.26 left). (2) You cannot reverse the sufficient direction with a boundary λ: a minimizer of λᵀz with λ on the boundary of K* need not be minimal (Fig 2.26 right).

Ex Bminimal but no minimum — each minimal point via some λ ≻ 0

Take S = {(1,3), (3,1), (2,2), (4,4)} (the §2.4.2 set with no minimum; the minimal set is {(1,3),(3,1),(2,2)}). Since there is no minimum, no single point wins for every λ ≻ 0 — but each minimal point is a minimizer for some suitable λ ≻ 0 (the sufficient direction).

(2,2) is a minimizer only in the tied λ=(1,1) case (one checks it minimizes λᵀz iff λ₁=λ₂), so no λ ≻ 0 makes it the unique minimizer — yet it is still minimal. The sufficient direction needs only "a minimizer," not the unique one; uniqueness is the extra strength that pins down a minimum, not a minimal element.

Example C — the sufficient direction, spelled outno convexity

Claim. If λ ≻K* 0 and x minimizes λᵀz over S, then x is minimal. Suppose not: some z ∈ S, z ≠ x has z ≼K x, i.e. x − z ≽K 0 and x − z ≠ 0. Pair with λ ≻K* 0:

\( \lambda^{\mathsf T}(x-z)>0 \quad\Longrightarrow\quad \lambda^{\mathsf T}z<\lambda^{\mathsf T}x. \)

But x minimizes λᵀz over S, so λᵀx ≤ λᵀz — a contradiction. Hence no such z; x is minimal. (No convexity used.)

Ex Dthe gap — a minimal point selected only by a boundary λ (Fig 2.26 left)

B&V's Figure 2.26 (left) shows a convex set S₁ ⊆ R² with a minimal point x₁ that is not a minimizer of λᵀz for any λ ≻ 0 — yet it is a minimizer for the boundary direction λ=(1,0) (a zero component, so λ ≽ 0 but not λ ≻ 0). The only supporting hyperplane touching at x₁ is vertical (normal (1,0)); every strictly-positive λ tilts the direction so its minimizer sits elsewhere on the boundary.

Fig 2.26 right shows the other side of the gap: a minimizer of λᵀz for a boundary λ=(0,1) that is not minimal — so the sufficient direction also cannot be relaxed to boundary λ.

§3example 2.27 — the Pareto optimal production frontier

A product needs n resources; a production method is a resource vector x ∈ Rⁿ (xᵢ ≥ 0 = amount of resource i used; less is better). The production set P ⊆ Rⁿ is all achievable resource vectors.

  • efficient
    The minimal elements of P (w.r.t. componentwise ≼) are the Pareto optimal methods; the set of them is the efficient production frontier.
  • better
    "Method x is better than y" ⟺ x ≼ y, x ≠ y — uses no more of any resource, and strictly less of at least one. Pareto optimal = no better method exists.
  • prices
    To find efficient methods, minimize λᵀx = λ₁x₁ + ⋯ + λₙxₙ over P with any λ ≻ 0. Here λᵢ is the price of resource i, so λᵀx is total cost — minimizing it finds the cheapest method, which for positive prices is guaranteed efficient (the sufficient direction).

This is the §2.4.2 "no minimum, several minimal" picture with an economic reading: usually there is no single best method, but a whole efficient frontier. When the cheapest-at-those-prices method exists, positive prices select an efficient one — but not every efficient method is reached by a strictly-positive price (Fig 2.27). That is the min-vs-minimal gap again.

Example E — Pareto production over a finite setlabor · fuel

Take P = {a=(2,5), b=(4,2), c=(5,6)} (labor, fuel). Pareto optimal = minimal in P under componentwise ≼:

a vs b: 2 < 4 but 5 > 2 — incomparable, neither dominates ⟹ both minimal. And c=(5,6) is dominated by both (a ≼ c and b ≼ c) ⟹ c not minimal. So the efficient methods are a and b; c is inefficient.

§4expert lens — what to keep straight (diagnose)

M1Minimum ⟺, clean

x minimum ⟺ for all λ ≻K* 0, x is the unique minimizer of λᵀz (strict supporting hyperplane; no convexity needed).

M2Minimal — sufficient

Always: λ ≻K* 0 and x minimizes λᵀz ⟹ x minimal. No convexity. Converse fails (Fig 2.25).

M3Minimal — necessary

Needs S convex: x minimal ⟹ some nonzero λ ≽K* 0 minimizes λᵀz — not strengthenable to λ ≻K* 0, and false without convexity.

M4Watch the quantifiers

A boundary-λ minimizer need not be minimal (Fig 2.26 right). Minimum = all λ + unique; minimal (sufficient) = some λ ≻K* 0.

M5Pareto reading

Minimal element of P = efficient. Minimize λᵀx over P with λ ≻ 0 (prices) to find one — but scalarization only reaches the supported efficient points.

Multi-objective framing. Minimizing a positive-weighted sum λᵀz is scalarization; when the minimization attains a minimizer, that minimizer is an efficient (Pareto) point — the sufficient direction. But scalarization only reaches the supported efficient points. Nonconvex problems can have unsupported Pareto points that no λ ≻ 0 reaches, and even in convex settings some minimal points require a boundary weight λ ≽ 0 rather than a strictly positive one. This is precisely the min-vs-minimal gap.

Main points for quick review

Minimum — clean iff

\(x\) minimum \(\iff\) for all \(\lambda\succ_{K^*}0\), \(x\) is the unique minimizer of \(\lambda^{\mathsf T}z\). Strict supporting hyperplane; no convexity needed.

Minimal — sufficient

\(\lambda\succ_{K^*}0\) & \(x\) minimizes \(\lambda^{\mathsf T}z\) \(\Rightarrow x\) minimal (always). Converse fails; needs only "a" minimizer, not unique.

Minimal — necessary

\(S\) convex & \(x\) minimal \(\Rightarrow\) some nonzero \(\lambda\succeq_{K^*}0\) minimizes \(\lambda^{\mathsf T}z\). Not strict, and false without convexity.

The gap

Boundary-\(\lambda\) minimizer need not be minimal (Fig 2.26 right); a minimal point may need boundary \(\lambda\) (left). Convexity closes only the necessary side.

Pareto (Ex 2.27)

Minimal in \(P\) = efficient. Minimize \(\lambda^{\mathsf T}x\), \(\lambda\succ0\) (prices) to find an efficient method — scalarization reaches only supported points.
