Dual generalized
inequalities

A proper cone K has a proper dual K*, so K* induces its own ordering. One cone relation x ≼K y becomes an infinite family of scalar tests λᵀx ≤ λᵀy — one per dual-feasible λ.

K — primal cone K* — dual cone & its λ every test agrees p. 53–54 · mastery: diagnose
§0where we are

§2.4.1 gave a proper cone K and its generalized inequality ≼K. §2.6.1 showed the dual cone K* of a proper cone is again proper. Put the two together: K* induces its own generalized inequality ≼K* — the dual of ≼K. The payoff is a clean way to test ≼K using linear functionals, plus a conic theorem of alternatives that generalizes the orthant case of §2.5.1.

§1the dual generalized inequality
Since K proper \(\Rightarrow\) K* proper (§2.6.1), the dual cone K* itself induces a generalized inequality \(\preceq_{K^*}\), called the dual of \(\preceq_K\).B&V §2.6.2 · L2751–2754

Nothing new is assumed — properness of K alone hands you a second proper cone, hence a second ordering, sitting in the same space of vectors that act as linear functionals on the first.

§2characterization — one inequality, a family of tests
\[ x \preceq_K y \iff \lambda^{\mathsf T}x \le \lambda^{\mathsf T}y \ \text{ for all } \lambda \succeq_{K^*} 0. \] \[ x \prec_K y \iff \lambda^{\mathsf T}x < \lambda^{\mathsf T}y \ \text{ for all } \lambda \succeq_{K^*} 0,\ \lambda \neq 0. \] B&V §2.6.2 · L2755–2759

Read it directly: x ≼K y holds exactly when every dual-feasible λ — every λ ≽K* 0 — agrees that λᵀx ≤ λᵀy. A single cone relation is equivalent to an infinite family of scalar tests, one per λ ∈ K*. It is the same "cone inequality = all of its dual linear tests" idea that defined the dual cone in the first place.

!

The family is infinite, but it is generated by the extreme rays of K*: if every extreme dual direction passes λᵀx ≤ λᵀy, so does every nonnegative combination. Checking the finitely many extreme rays certifies the whole ordering.

§3swap symmetry — the double dual returns
Because (K*)* = K for proper cones (§2.6.1), the dual of \(\preceq_{K^*}\) is \(\preceq_K\) again. So the characterization holds with the roles swapped: \[ \lambda \preceq_{K^*} \mu \iff \lambda^{\mathsf T}x \le \mu^{\mathsf T}x \ \text{ for all } x \succeq_K 0. \]B&V §2.6.2 · L2761–2763

≼K and ≼K* are a symmetric pair — each tests the other. Testing a primal relation ranges λ over K*; testing a dual relation ranges x over K. The double-dual identity is exactly what makes the two directions mirror images.

§4example 2.26 — a conic theorem of alternatives

Let K ⊆ Rᵐ be proper and consider the strict inequality \[ Ax \prec_K b. \tag{2.21}\] When is it infeasible? The dual generalized inequality supplies a certificate.

Infeasible \(\Rightarrow\) a certificate in K*separation, then read off K*
\[ \{\,b-Ax\,\}\ \cap\ \operatorname{int}K=\varnothing \ \Rightarrow\ \exists\,\lambda\neq0,\mu:\ \lambda^{\mathsf T}(b-Ax)\le\mu\ \forall x,\quad \lambda^{\mathsf T}y\ge\mu\ \forall y\in\operatorname{int}K \]
infeasible means the affine set \(\{b-Ax\}\) misses the open set \(\operatorname{int}K\); separate them
\[ \lambda^{\mathsf T}(b-Ax)\le\mu\ \forall x \ \Rightarrow\ A^{\mathsf T}\lambda=0,\quad \lambda^{\mathsf T}b\le\mu \]
a linear function of \(x\) bounded above on all of \(\mathbb R^n\) has zero coefficient
\[ \lambda^{\mathsf T}y\ge\mu\ \forall y\in K \ \Rightarrow\ \lambda\succeq_{K^*}0,\quad \mu\le0 \]
\(\lambda^{\mathsf T}y\) bounded below on the cone \(K\) forces λ ∈ K*, and \(y=0\) gives \(\mu\le0\)
\[ \boxed{\,\lambda\neq0,\quad \lambda\succeq_{K^*}0,\quad A^{\mathsf T}\lambda=0,\quad \lambda^{\mathsf T}b\le0\,} \tag{2.22}\]
combine: \(\lambda^{\mathsf T}b\le\mu\le0\). The certificate lives in the dual cone, not K
↺

Certificate \(\Rightarrow\) infeasible. If both held, take any x with b − Ax ≻K 0. The strict characterization gives λᵀ(b − Ax) > 0 (since λ ≠ 0). But Aᵀλ = 0 makes λᵀ(b − Ax) = λᵀb ≤ 0 — a contradiction. So (2.21) and (2.22) can't both hold.

Together the two directions make (2.21) and (2.22) a pair of alternatives: exactly one is feasible.

System I — primal (2.21)

Ax ≺K b — find x putting b − Ax strictly inside the cone K.

System II — certificate (2.22)

λ ≠ 0, λ ≽K* 0, Aᵀλ = 0, λᵀb ≤ 0 — a nonzero dual-cone λ in the null space of Aᵀ with λᵀb ≤ 0.

!

Set K = Rᵐ₊ and ≽K* becomes componentwise ≽ 0: this is exactly the §2.5.1 orthant theorem of alternatives. Example 2.26 is its conic generalization — a Farkas lemma over an arbitrary proper cone.

§5expert lens — diagnose

G1The dual ordering

K proper \(\Rightarrow\) K* proper \(\Rightarrow\) the dual generalized inequality ≼K*. No new hypotheses beyond properness of K.

G2Characterization

x ≼K y ⟺ λᵀx ≤ λᵀy for all λ ≽K* 0 (strict: <, λ ≠ 0). One cone inequality = the full family of dual linear tests; extreme rays of K* suffice.

G3Swap symmetry

(K*)* = K \(\Rightarrow\) the dual of ≼K* is ≼K. e.g. λ ≼K* µ ⟺ λᵀx ≤ µᵀx ∀ x ≽K 0. A symmetric pair.

G4Alternatives

Ax ≺K b infeasible ⟺ ∃ λ ≠ 0, λ ≽K* 0, Aᵀλ = 0, λᵀb ≤ 0. Exactly one system feasible; the certificate lives in K*. Generalizes the §2.5.1 orthant case (conic Farkas).

Main points for quick review

Dual ordering

\(K\) proper \(\Rightarrow K^*\) proper \(\Rightarrow\) the dual generalized inequality \(\preceq_{K^*}\).

Characterization

\(x\preceq_K y \iff \lambda^{\mathsf T}x\le\lambda^{\mathsf T}y\) for all \(\lambda\succeq_{K^*}0\) (strict: \(<\), \(\lambda\neq0\)). One cone inequality = the full family of dual linear tests.

Swap symmetry

\((K^*)^*=K \Rightarrow\) the dual of \(\preceq_{K^*}\) is \(\preceq_K\), e.g. \(\lambda\preceq_{K^*}\mu \iff \lambda^{\mathsf T}x\le\mu^{\mathsf T}x\ \forall x\succeq_K 0\).

Alternatives (Ex. 2.26)

\(Ax\prec_K b\) infeasible \(\iff \exists\,\lambda\neq0,\lambda\succeq_{K^*}0,A^{\mathsf T}\lambda=0,\lambda^{\mathsf T}b\le0\) — exactly one feasible; certificate in \(K^*\). Conic Farkas.
