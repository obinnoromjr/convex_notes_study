§0the calculus of convex sets

§2.2 gave a stock of convex sets — halfspaces, balls, cones, polyhedra, the PSD cone. §2.3 starts the other half: operations that preserve convexity. Together they form a calculus of convex sets — you establish convexity of a new set by building it from known ones, instead of returning to the definition each time. Intersection is the first move.

§1intersection preserves convexity
If \(S_1,S_2\) are convex then \(S_1\cap S_2\) is convex. This extends to arbitrary (even infinite) families: if \(S_\alpha\) is convex for every \(\alpha\in\mathcal A\), then \(\bigcap_{\alpha\in\mathcal A} S_\alpha\) is convex.B&V §2.3.1 · L1851
Proof — one linethe segment test, everywhere at once
\[ x,y\in\textstyle\bigcap_\alpha S_\alpha,\ \theta\in[0,1]\ \Rightarrow\ x,y\in S_\alpha\ \text{for every }\alpha \]
"in the intersection" = "in every set at once"
\[ \Rightarrow\ \theta x+(1-\theta)y\in S_\alpha\ \text{for every }\alpha \]
each \(S_\alpha\) convex traps the whole segment in it
\[ \Rightarrow\ \theta x+(1-\theta)y\in\textstyle\bigcap_\alpha S_\alpha. \quad\blacksquare \]
in every \(S_\alpha\) ⟹ back in the intersection
≠

Why unions are different. The union of convex sets need not be convex: take two disjoint disks — a point in each has a connecting segment that leaves the union through the gap. Intersection traps the segment in every set; union only needs it in one, so it can slip out. Intersection ✓, union ✗.

First example (re-deriving §2.2.4): a polyhedron {x | Ax ≼ b, Cx = d} is a finite intersection of halfspaces and hyperplanes, each convex — so it's convex. A one-liner now.

§2example 2.7 — the PSD cone, again
\(S^n_+=\bigcap_{z\neq0}\{X\in S^n \mid z^{\mathsf T}Xz \ge 0\}\).B&V Example 2.7 · L1858

The proof we pointed forward to from §2.2.5. Read it through the PSD definition: X ≽ 0 means zᵀXz ≥ 0 for every z. So "X is PSD" = "X lies in every set {X | zᵀXz ≥ 0}" = "X is in the intersection." The intersection over z assembles the for-all in the definition.

Each piece is a halfspace. Fix z ≠ 0: the map X ↦ zᵀXz = Σᵢⱼ zᵢzⱼXᵢⱼ = ⟨zzᵀ, X⟩ is a linear function of X (not identically zero), so {X | zᵀXz ≥ 0} is a halfspace in Sⁿ. An intersection of infinitely many halfspaces ⟹ convex.

!

Careful: linear in X, quadratic in z. The halfspace structure lives in X (the variable); z just indexes which halfspace. And it's an infinite intersection — a second, independent proof that Sⁿ₊ is convex.

§3example 2.8 — a continuum of slabs
\(S=\{x\in\mathbf R^m \mid |p(t)|\le 1\ \text{for all } |t|\le \pi/3\}\), where \(p(t)=\sum_{k=1}^m x_k\cos kt\).B&V Example 2.8 · L1873

Subtler: the variable is a coefficient vector, the constraints indexed by a continuous parameter t. Fix t — then p(t) = (cos t, cos 2t, …, cos mt)ᵀx is linear in x (the cosines are fixed numbers), so |p(t)| ≤ 1 is a slab Sₜ = {x | −1 ≤ (cos t,…)ᵀx ≤ 1}. The full set is S = ∩_{|t|≤π/3} Sₜ — infinitely many slabs ⟹ convex.

★

The reframing that matters: a constraint that looks nonlinear "in t" is linear in the decision variable x for each fixed t; t merely indexes the family. A requirement holding for a continuum of t (a semi-infinite constraint) is an infinite intersection — still convex.

§4the converse — forward to §2.5.1

Every closed convex set S is a (usually infinite) intersection of halfspaces — in fact of all halfspaces containing it:

\[ S=\bigcap\{H \mid H\text{ a halfspace},\ S\subseteq H\}. \]
↺

So "intersection of halfspaces" isn't just a way to build convex sets — for closed sets it is convexity. (The "closed" qualifier matters; this is proved in §2.5.1 via separating hyperplanes — not today.)

§5expert lens — diagnose

D1The rule, in full

Intersection preserves convexity — finite and arbitrary/infinite (∩_{α∈A} Sα convex if every Sα is). Subspaces, affine sets, convex cones likewise. Union fails.

D2The standard move

To prove a set convex, write it as an intersection of convex sets — usually halfspaces or slabs.

D3PSD via intersection

Sⁿ₊ = ∩_{z≠0}{X | zᵀXz ≥ 0}; zᵀXz = ⟨zzᵀ,X⟩ linear in X → each a halfspace (linear in X, quadratic in z; infinite ∩).

D4Semi-infinite constraints

For fixed t the constraint is linear in x (a slab); the for-all-t requirement is an infinite intersection — still convex.

Main points for quick review

The rule

Convexity is preserved under intersection — finite AND arbitrary/infinite: \(\bigcap_{\alpha} S_\alpha\) convex if every \(S_\alpha\) is. Subspaces / affine / cones too. Union does NOT (two disjoint disks).

Proof engine

x, y in the intersection ⟹ in every \(S_\alpha\) ⟹ segment in every \(S_\alpha\) ⟹ in the intersection. Standard move: write the set as an intersection of halfspaces / slabs.

Examples 2.7 / 2.8

\(S^n_+=\bigcap_{z\neq0}\{X\mid z^{\mathsf T}Xz\ge0\}\) (each a halfspace in X). \(|p(t)|\le1\ \forall t\) = infinite ∩ of slabs (linear in x for fixed t).

Converse (forward)

§2.5.1: every closed convex set is an intersection of halfspaces — for closed sets, that IS convexity.
