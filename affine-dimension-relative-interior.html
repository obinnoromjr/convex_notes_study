Affine dimension
& relative interior

How big is the flat a set lives in — and where is its “inside” once you stop measuring against all of \mathbb{R}^n and start measuring inside that flat?

builds on §2.1.2 · aff C, dim affine dim C := dim(aff C) relint = interior inside aff C
§0

Where we are

§2.1.2 handed us two tools: the affine hull \operatorname{aff} C (the smallest flat containing C) and the dimension of an affine set. This section spends both at once.

It defines the affine dimension of any set, and then its relative interior — the interior computed inside the flat the set lives on, not inside all of \mathbb{R}^n. That second idea is load-bearing: it is the fix for the embarrassing fact that lower-dimensional sets have empty ordinary interior.

§1

Affine dimension

The definition is a one-liner that just composes the two pieces from §2.1.2:

Definition · affine dimension
\operatorname{affine\,dim} C \;:=\; \dim(\operatorname{aff} C)

The dimension of the carrying flat — the smallest affine set C sits inside.

B&V attach a warning: this “is not always consistent with other definitions of dimension.” The example to memorize is the unit circle.

Why aff = ℝ². The circle contains three non-collinear points (red); by §2.1.2 their affine hull is the entire plane. So affine dim = 2, even though the curve is topologically 1-dimensional. Affine dimension measures the smallest flat C sits in — it cannot see how C curves within that flat.
§2

Relative interior

The motivation. If \operatorname{affine\,dim} C < n, then C lives in \operatorname{aff} C \neq \mathbb{R}^n. Every \mathbb{R}^n-ball drawn around a point of C immediately pokes off the flat — so it can never fit inside C. A segment has no 2-D interior; a flat square has no 3-D interior. Ordinary interior is empty, and that is useless.

The fix: measure the interior relative to the flat. Keep only the slice of the ball that lies in \operatorname{aff} C.

Definition · relative interior
\operatorname{relint} C \;=\; \{\, x \in C \;:\; B(x,r)\cap \operatorname{aff} C \subseteq C \ \text{ for some } r>0 \,\}

where B(x,r) is the ball in any norm. The whole point is the ∩ aff C: require only the in-flat slice of the ball to stay in C — not the full \mathbb{R}^n-ball (that would be ordinary interior, which is empty here).

Same point, same ball — two questions. On the left the full 2-D ball must fit in the 1-D segment: impossible, so int C = ∅. On the right we keep only B(x,r) ∩ aff C (the green chord on the line). It stays inside C, so x ∈ relint C. The ∩ aff C does not merely shrink the radius — it drops the off-flat directions entirely.
Definition · relative boundary
\operatorname{relbd} C \;=\; \operatorname{cl} C \,\setminus\, \operatorname{relint} C

Closure minus relative interior — the exact analogue of “boundary = closure − interior,” but with relint in place of int.

Don’t over-read
relint is not automatically nonempty. It is nonempty only when C is “solid” within \operatorname{aff} C — true for the segment and the square, false for the circle, whose aff hull is the full plane, so \operatorname{relint} C = \varnothing.
§3

Example 2.2 — the flat square in ℝ³

C = \{\, x \in \mathbb{R}^3 \;:\; -1 \le x_1 \le 1,\; -1 \le x_2 \le 1,\; x_3 = 0 \,\}
One picture, four answers. The blue sheet is aff C = {x₃ = 0} (affine dim 2). The grey ball shows why int C = ∅ in ℝ³ — it always catches points with x₃ ≠ 0. The green fill is relint C, the open square. The coral wireframe is the relative boundary.

relbd C = cl C − relint C. The closed square splits cleanly into the open square (relint) and the rim (relative boundary). Note this uses relint, not int — the ordinary boundary here would be the whole square, which tells you nothing.
§4

Expert lens — how to diagnose

Four traps separate “recognize” from “diagnose”:

Three “dimensions” that disagree

NotionWhat it measuresCircle
ambientthe space C sits in2
affine dimdim(aff C) — smallest flat2
topologicallocal manifold dimension1

Affine dim is none of the other two by default. The circle pins all three apart.

The two everyone gets wrong

1. relint uses \cap \operatorname{aff} C — it restricts the ball to the flat, it does not merely shrink the radius. And relint may still be empty (the circle).

2. Relative boundary is \operatorname{cl} C \setminus \operatorname{relint} C — minus relint, not minus int. For flat sets these differ: the ordinary boundary is degenerate (= cl C), while the relative boundary is the genuine rim.

The procedure: compute from the definition

Always derive, don’t pattern-match. Find aff C; get relint from B(x,r) ∩ aff C ⊆ C; then relbd = cl C − relint. The “take the strict inequalities / open version” shortcut works for tidy closed-box sets like the segment and square — but that is a feature of those examples, not a general rule.

Quick review

  • Affine dimension = \dim(\operatorname{aff} C) — the smallest flat containing C. Can exceed topological dim: unit circle → affine 2, topological 1.
  • relint C = \{x\in C : B(x,r)\cap \operatorname{aff} C \subseteq C\}. The ∩ aff C is the whole point — not a smaller ℝⁿ ball.
  • Why relint exists: a set of affine dim < n has empty ordinary interior in ℝⁿ. relint repairs it — when C is solid in its flat. It can still be \varnothing (the circle).
  • Relative boundary = \operatorname{cl} C \setminus \operatorname{relint} C  (minus relint, not int).
  • Example 2.2 (flat square): aff = {x₃ = 0}, int = ∅, relint = open square, relbd = wireframe.
  • Compute from the definition; the strict-inequality shortcut is only for tidy closed-box sets.
