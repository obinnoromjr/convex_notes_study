The epigraph

The exact bridge from functions to sets: a function is convex if and only if the region on and above its graph is a convex set. Unlike sublevel sets, the epigraph loses nothing — it lets Chapter 2's set machinery (supporting hyperplanes, LMIs) speak directly about functions.

the graph — lower boundary epi f — on/above f convex ⟺ epi f convex p. 75–77 · mastery: diagnose
§0where we are

§3.1.6 related convex functions to convex sets one way: a convex function has convex sublevel sets, but the converse fails (−eˣ has convex sublevel sets yet isn't convex). §3.1.7 gives the exact bridge with no such gap — a function is convex if and only if its epigraph is a convex set. This is the link that lets you import Chapter 2's set machinery — supporting hyperplanes, LMIs — directly into statements about functions.

§1graph & epigraph

The graph of a function f : Rⁿ → R is {(x, f(x)) | x ∈ dom f} ⊆ Rⁿ⁺¹. The epigraph is everything on or above it:

\[ \operatorname{epi}f=\{(x,t)\mid x\in\operatorname{dom}f,\ f(x)\le t\}\subseteq\mathbb{R}^{n+1}. \]B&V §3.1.7 · L3821–3827

“Epi” means “above” — every point on or above the graph. The lower boundary of epi f is exactly the graph. The mirror object is the hypograph, everything on or below the graph:

\[ \operatorname{hypo}f=\{(x,t)\mid t\le f(x)\}. \]B&V §3.1.7 · L3828–3832

§2the link — a genuine biconditional
A function is convex if and only if its epigraph is a convex set. A function is concave if and only if its hypograph is a convex set.B&V §3.1.7 · L3828–3832

This is the payoff. Recall the contrast with §3.1.6: sublevel-set convexity is only one-directional — the converse fails (−eˣ). The epigraph statement has no such gap. Sublevel sets throw information away; the epigraph keeps all of it, so it is necessary and sufficient. When you want to prove a function convex from a fact about sets, the epigraph is the tool — sublevel sets cannot do it.

Proof — both directionsmembership ⇄ (3.1)
\[ (x,t),(y,s)\in\operatorname{epi}f\ \Rightarrow\ f(x)\le t,\ f(y)\le s \]
(⟹) convexity gives \(f(\theta x+(1-\theta)y)\le\theta f(x)+(1-\theta)f(y)\le\theta t+(1-\theta)s\), so \(\theta(x,t)+(1-\theta)(y,s)\in\operatorname{epi}f\)
\[ (x,f(x)),(y,f(y))\in\operatorname{epi}f\ \text{convex} \]
(⟸) the two graph points combine to \(\big(\theta x+(1-\theta)y,\ \theta f(x)+(1-\theta)f(y)\big)\in\operatorname{epi}f\)
\[ \Rightarrow\ f(\theta x+(1-\theta)y)\le\theta f(x)+(1-\theta)f(y) \]
membership unpacks to \(\operatorname{dom}f\) convex and exactly the Jensen inequality (3.1). ∎
Worked · epi(x²)t ≥ x²

\(\operatorname{epi}(x^2)=\{(x,t)\mid t\ge x^2\}\) — the filled region on and inside the parabola. It is convex: take \((x,t),(y,s)\) with \(t\ge x^2,\ s\ge y^2\); then along the chord \(\theta t+(1-\theta)s\ge\theta x^2+(1-\theta)y^2\ge(\theta x+(1-\theta)y)^2\), the last step being convexity of \(x^2\). Convex epigraph ⟺ convex function, seen concretely.

Worked · max as an intersection of halfspacesepi(max fᵢ) = ⋂ epi fᵢ

For \(f(x)=\max_i a_i^{\mathsf T}x+b_i\), the condition \(f(x)\le t\) means every \(a_i^{\mathsf T}x+b_i\le t\) at once, so \(\operatorname{epi}f=\bigcap_i\{(x,t)\mid a_i^{\mathsf T}x+b_i\le t\}\) — an intersection of halfspaces in \((x,t)\). Each halfspace is convex, the intersection is convex, hence \(f\) is convex. The epigraph turns “max of affine” into pure Chapter-2 set algebra.

§3Example 3.4 — the matrix fractional function

The epigraph is not just a picture — it is a proof device. Take

\[ f(x,Y)=x^{\mathsf T}Y^{-1}x\quad\text{on}\quad \operatorname{dom}f=\mathbb{R}^n\times\mathbb{S}^n_{++}, \]B&V §3.1.7 · Ex 3.4 · L3846–3874

which generalizes \(x^2/y\) from §3.1.5. Writing its epigraph and applying the Schur complement (§A.5.5, used not developed):

\[ \operatorname{epi}f=\Big\{(x,Y,t)\ \Big|\ Y\succ0,\ x^{\mathsf T}Y^{-1}x\le t\Big\}=\Big\{(x,Y,t)\ \Big|\ \begin{bmatrix}Y&x\\ x^{\mathsf T}&t\end{bmatrix}\succeq0,\ Y\succ0\Big\}. \]B&V §3.1.7 · Ex 3.4

The block constraint \(\left[\begin{smallmatrix}Y&x\\x^{\mathsf T}&t\end{smallmatrix}\right]\succeq0\) is an LMI: affine in \((x,Y,t)\), so the set it defines is the inverse image of the convex PSD cone \(\mathbb{S}^{n+1}_+\) under an affine map — convex. Intersect with \(\{Y\succ0\}\) (also convex) and epi f is convex as an intersection ⟹ f is convex.

!

The \(Y\succ0\) conjunct isn't decoration. The Schur equivalence only holds for \(Y\succ0\); the block-PSD set alone is strictly larger. At \(n=1\) it admits \((x,y,t)=(0,0,1)\) — the matrix \(\left[\begin{smallmatrix}0&0\\0&1\end{smallmatrix}\right]\succeq0\) — which lies outside \(\operatorname{dom}f\). And \(Y\succ0\) is a strict inequality, so it is not itself an LMI.

The scalar case n = 1 is exactly \(x^2/y\): the LMI reads \(\left[\begin{smallmatrix}y&x\\x&t\end{smallmatrix}\right]\succeq0\) with \(y>0\), i.e. \(yt-x^2\ge0\). That is the lab below.

§4first-order condition, read on the epigraph

For differentiable convex f, if (y,t) ∈ epi f then t ≥ f(y) ≥ f(x) + ∇f(x)ᵀ(y−x). Rearranging puts the gradient inequality into hyperplane form:

\[ (y,t)\in\operatorname{epi}f\ \Longrightarrow\ \big(\nabla f(x),-1\big)^{\mathsf T}\!\big[(y,t)-(x,f(x))\big]\le0. \]B&V §3.1.7 · L3877–3915

Read it geometrically: the hyperplane with normal (∇f(x), −1) supports epi f at the boundary point (x, f(x)) (Fig 3.6). So the first-order condition (§3.1.3) is the supporting-hyperplane statement (§2.5.2) applied to epi f — the gradient inequality and the supporting hyperplane are the same fact seen from two sides.

Worked · supporting hyperplane to epi(x²) at x = 1normal (2, −1)

For \(f(x)=x^2\), \(\nabla f(1)=2\), and the graph point is \((1,1)\). The supporting hyperplane of \(\operatorname{epi}f\) at \((1,1)\) has normal \((2,-1)\): \((2,-1)^{\mathsf T}\big[(y,t)-(1,1)\big]=2(y-1)-(t-1)\le0\), i.e. \(t\ge 2y-1\). That is precisely the tangent line \(t=1+2(y-1)\) — the graph’s tangent at \(x=1\), lying under the whole epigraph.

§5expert lens — diagnose

D1The objects

epi f = {(x,t) | x ∈ dom f, f(x) ≤ t} ⊆ Rⁿ⁺¹ — on or above the graph. hypo f uses t ≤ f(x).

D2The biconditional

f convex ⟺ epi f convex — genuine both ways. f concave ⟺ hypo f convex.

D3Why not sublevel sets

Sublevel convexity is only necessary (−eˣ); the epigraph is necessary and sufficient. To prove convexity from a set fact, reach for the epigraph.

D4Epigraph as proof device

Example 3.4 writes it as the LMI [[Y,x],[xᵀ,t]] ⪰ 0 (affine preimage of the PSD cone, convex) ∩ {Y ≻ 0} (convex) ⟹ convex. The Y ≻ 0 can't be dropped — it isn't an LMI (strict).

D5First-order = supporting hyperplane

(∇f(x), −1) is the normal of a hyperplane supporting epi f at (x, f(x)) — the gradient inequality is that support statement.

Summary · 2026-07-16 · bv.3.1.7

Main points for quick review

Definition

\(\operatorname{epi}f=\{(x,t)\mid x\in\operatorname{dom}f,\ f(x)\le t\}\subseteq\mathbb{R}^{n+1}\) (on/above the graph); \(\operatorname{hypo}f\) uses \(t\le f(x)\).

The biconditional

\(f\) convex \(\iff\operatorname{epi}f\) convex — genuine both ways (contrast §3.1.6's one-directional sublevel result). \(f\) concave \(\iff\operatorname{hypo}f\) convex.

Proof

(⟹) combine two epigraph points; (⟸) apply convexity of \(\operatorname{epi}f\) to the two graph points — membership yields both \(\operatorname{dom}f\) convex and (3.1).

Example 3.4

\(x^{\mathsf T}Y^{-1}x\) convex on \(\mathbb{R}^n\times\mathbb{S}^n_{++}\): \(\operatorname{epi}f=\) the Schur LMI \(\left[\begin{smallmatrix}Y&x\\x^{\mathsf T}&t\end{smallmatrix}\right]\succeq0\ \cap\ \{Y\succ0\}\). \(n=1\) recovers \(x^2/y\).

First-order (Fig 3.6)

\((\nabla f(x),-1)\) is the normal of a hyperplane supporting \(\operatorname{epi}f\) at \((x,f(x))\) — the first-order condition is a supporting-hyperplane statement.
