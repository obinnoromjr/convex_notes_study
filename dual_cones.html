§0where we are

§2.4 built generalized inequalities from proper cones. §2.6 takes the dual view. The dual cone of a cone K is the set of directions that make a nonnegative inner product with everything in K — built directly from the supporting hyperplanes at the origin from §2.5.2, and the backbone of conic-programming duality.

§1the dual cone
\[ K^{*}=\{\,y \mid x^{\mathsf T}y\ge 0 \ \text{for all } x\in K\,\}. \tag{2.19}\]B&V §2.6.1 · L2632–2636

In words: y ∈ K* exactly when y makes a nonnegative inner product with every point of K. The name is apt — K* is itself a cone. Two facts do the heavy lifting:

  • cone
    K* is a cone: if y∈K* and t≥0 then xᵀ(ty)=t(xᵀy)≥0, so ty∈K*.
  • closed & convex
    K* is always convex and closed — even when K is neither. Because \(K^{*}=\bigcap_{x\in K}\{y \mid x^{\mathsf T}y\ge 0\}\), an intersection of closed halfspaces (one per x∈K), and any intersection of closed convex sets is closed and convex.
!

Watch the sign and the quantifier: it is xᵀy ≥ 0 (not ≤ 0), and it must hold for all x∈K. Flip either and you have a different object.

§2geometric interpretation

y ∈ K* iff −y is the normal of a hyperplane that supports K at the origin. Equivalently, the halfspace {z | yᵀz ≥ 0} — the one with inward normal y — contains all of K. So the dual cone collects the inward normals of every halfspace through 0 that contains K: exactly the §2.5.2 supporting-hyperplane idea, applied at the origin (Figure 2.22).

§3examples
Example 2.22 — subspace → orthogonal complementV* = V⊥

A subspace V ⊆ Rⁿ (a cone) has dual V* = V⊥ = {y | vᵀy = 0 ∀ v ∈ V}, its orthogonal complement.

Why "= 0", not "≥ 0": for x∈V, also −x∈V, so membership needs both xᵀy ≥ 0 and (−x)ᵀy ≥ 0, which force xᵀy = 0. A subspace's dual is generally not itself.

Example 2.23 — nonnegative orthant (self-dual)(Rⁿ₊)* = Rⁿ₊

The nonnegative orthant is self-dual: xᵀy ≥ 0 for all x ≽ 0 ⟺ y ≽ 0. (A cone with K* = K is called self-dual.) The compute move is to test the generators x = eᵢ: eᵢᵀy = yᵢ ≥ 0 for each i is exactly y ≽ 0.

Example 2.24 — positive semidefinite cone (self-dual)(Sⁿ₊)* = Sⁿ₊

With the trace inner product ⟨X,Y⟩ = tr(XY), the PSD cone is self-dual: tr(XY) ≥ 0 for all X ≽ 0 ⟺ Y ≽ 0. Both directions have clean witnesses.

Proof — both inclusionsrank-one witness ↔ eigendecomposition
\[ Y\notin S^n_+ \Rightarrow \exists q,\ q^{\mathsf T}Yq<0 \]
witness: since \(q^{\mathsf T}Yq=\operatorname{tr}(qq^{\mathsf T}Y)\), the PSD matrix \(X=qq^{\mathsf T}\) gives \(\operatorname{tr}(XY)<0\), so \(Y\notin(S^n_+)^{*}\)
\[ X,Y\succeq0:\quad X=\textstyle\sum_i\lambda_i\,q_iq_i^{\mathsf T},\ \lambda_i\ge0 \]
eigendecompose the PSD matrix \(X\); the \(\lambda_i\) are its nonnegative eigenvalues
\[ \operatorname{tr}(YX)=\sum_i\lambda_i\,(q_i^{\mathsf T}Yq_i)\ge0 \]
each \(q_i^{\mathsf T}Yq_i\ge0\) because \(Y\succeq0\); nonneg combo ⇒ \(Y\in(S^n_+)^{*}\). ∎
Example 2.25 — norm cone → dual-norm cone‖u‖* ≤ v

The norm cone K = {(x,t) | ‖x‖ ≤ t} has dual K* = {(u,v) | ‖u‖* ≤ v}, where the dual norm is ‖u‖* = sup{uᵀx | ‖x‖ ≤ 1}. The heart is condition (2.20):

\[ x^{\mathsf T}u+tv\ge0 \ \text{for all }(x,t)\text{ with }\|x\|\le t \iff \|u\|_{*}\le v. \tag{2.20}\]B&V §2.6.1 · L2706–2729

Since the ℓ₂ norm is self-dual (‖·‖* = ‖·‖₂), the second-order cone is self-dual.

§4properties
  • a
    K* is closed and convex — always, for any set K.
  • b
    Order-reversing: K₁ ⊆ K₂ ⟹ K₂* ⊆ K₁*. Bigger cone, smaller dual.
  • c
    int K ≠ ∅ ⟹ K* is pointed (contains no line).
  • d
    cl K pointed ⟹ K* is solid (has nonempty interior).
  • e
    K** = cl conv K — the second dual is the closed convex hull. So K** = K whenever K is convex and closed.
▸

Consequence. If K is a proper cone (closed, convex, solid, pointed), then so is K*, and K** = K. Duality is an involution on proper cones — apply it twice, get back where you started.

§5expert lens — diagnose

D1The definition

K* = {y | xᵀy ≥ 0 ∀ x∈K} — sign ≥ 0 (not ≤), quantifier for all x. Always convex and closed (a halfspace intersection).

D2Geometric read

y∈K* ⟺ halfspace {yᵀz ≥ 0} contains K ⟺ −y normals a hyperplane supporting K at the origin.

D3Example roster

subspace → orthogonal complement V⊥; orthant & PSD cone self-dual; norm cone → dual-norm cone (ℓ₂ / SOC self-dual).

D4Properties

closed/convex; order-reversing; int K≠∅ ⟹ pointed; K** = K for proper K — dual of a proper cone is proper.

D5Compute move

Ask "which y satisfy xᵀy ≥ 0 for every x∈K?" — orthant: test x = eᵢ; subspace: use ±x ∈ V; finitely-generated: test the extreme rays.

Main points for quick review

Definition

\(K^{*}=\{y\mid x^{\mathsf T}y\ge0\ \forall x\in K\}\) — always convex and closed (intersection of halfspaces), even if \(K\) isn't.

Geometric

\(y\in K^{*}\) iff the halfspace with inward normal \(y\) contains \(K\), iff \(-y\) normals a hyperplane supporting \(K\) at the origin.

Examples

subspace → orthogonal complement \(V^{\perp}\); \(\mathbf R^n_+\) and \(S^n_+\) (trace product) self-dual; norm cone → dual-norm cone (ℓ₂ self-dual ⟹ SOC self-dual).

Properties

closed/convex; order-reversing \((K_1\subseteq K_2\Rightarrow K_2^{*}\subseteq K_1^{*})\); \(\operatorname{int}K\neq\emptyset\Rightarrow K^{*}\) pointed; \(K^{**}=\operatorname{cl\,conv}K\), so \(K^{**}=K\) for proper \(K\) — and the dual of a proper cone is proper.
