§0where we are

§2.2–§2.4 built and manipulated convex sets. §2.5 is the first big existence theorem: two convex sets that don't overlap can always be split by a hyperplane. This is the geometric engine behind LP/convex duality and theorems of alternatives.

§1the theorem & the construction
If \(C,D\) are nonempty disjoint convex sets, there exist \(a\neq0\) and \(b\) with \(a^{\mathsf T}x\le b\) for all \(x\in C\) and \(a^{\mathsf T}x\ge b\) for all \(x\in D\).B&V §2.5.1 · L2382

The affine function aᵀx − b is ≤ 0 on C, ≥ 0 on D; the hyperplane {aᵀx = b} separates them. Note what's required — just nonempty, disjoint, convex. No closedness or boundedness for the non-strict version.

The closest-points constructionthis is the design move
\[ a = d-c,\qquad b=\tfrac{1}{2}(\|d\|_2^2-\|c\|_2^2),\qquad f(x)=a^{\mathsf T}x-b=(d-c)^{\mathsf T}\!\big(x-\tfrac12(d+c)\big) \]
c ∈ C, d ∈ D the attained closest points; the plane bisects [c,d] perpendicularly
\[ \text{if some } u\in D \text{ had } f(u)<0:\ \ (d-c)^{\mathsf T}(u-d)<0 \]
rewrite \(f(u)=(d-c)^{\mathsf T}(u-d)+\tfrac12\|d-c\|_2^2\)
\[ \tfrac{d}{dt}\|d+t(u-d)-c\|_2^2\big|_0=2(d-c)^{\mathsf T}(u-d)<0 \]
convexity of D keeps \(d+t(u-d)\in D\), yet it's closer to \(c\) than \(d\) — contradicts \(d\) closest. ∎
!

Get the construction right: a = d − c (not c + d), and the plane passes through the midpoint ½(c+d) — not through c or d. It is perpendicular to the segment [c,d].

§2example 2.19 — separating from an affine set

Take C convex and D = {Fu + g | u ∈ Rᵐ} an affine set, disjoint from C. Separation gives a ≠ 0 with aᵀ(Fu + g) ≥ b for all u, i.e. aᵀFu ≥ b − aᵀg for every u ∈ Rᵐ. But a linear function of u bounded below on all of Rᵐ must be identically zero — so its coefficient vanishes:

\[ \exists\, a\neq0 \ \text{ with }\ F^{\mathsf T}a = 0 \ \text{ and }\ a^{\mathsf T}x \le a^{\mathsf T}g \ \text{ for all } x\in C \qquad(\text{and } b\le a^{\mathsf T}g). \]
↳

The separating normal a is forced orthogonal to the range of F — the directions the affine set is free to move in. Every "separate a convex set from a subspace / affine set" argument takes this shape.

§3strict separation — a point & a closed convex set

Strict separation means aᵀx < b on C, aᵀx > b on D. It's not always achievable for disjoint convex sets — even both closed (think of two sets approaching each other asymptotically). But a point and a disjoint closed convex set always can be strictly separated (Example 2.20): take a small ball around the point disjoint from C, separate, then shift the offset.

↺

The payoff (§2.3.1 converse, now proved): a closed convex set is the intersection of all halfspaces containing it — because if x ∉ C, strict separation hands you a halfspace that contains C but excludes x.

§4the converse — one set must be open

Does "a separating hyperplane exists" imply the sets are disjoint? In general, no. Take C = D = {0}: the hyperplane {x = 0} "separates" them — aᵀx ≤ 0 on C and aᵀx ≥ 0 on D both hold — yet they are not disjoint. The converse needs one extra hypothesis.

If C, D are convex, C is open, and a nonconstant affine f (a ≠ 0) satisfies f ≤ 0 on C and f ≥ 0 on D, then C and D are disjoint.B&V §2.5.1 · L2546

Putting the theorem and this together: two convex sets, at least one open, are disjoint if and only if a separating hyperplane exists. This "open-set" form is exactly what the theorem of alternatives below leans on (D = Rᵐ₊₊ is open).

§5example 2.21 — theorem of alternatives

When is Ax ≺ b (strict) infeasible? Iff C = {b − Ax} (affine) and D = Rᵐ₊₊ (open) don't meet. By the open-set converse, iff a separating hyperplane exists. Simplifying gives a clean certificate — and the two systems are a pair of alternatives: for any A, b, exactly one is solvable.

System I — primal

Ax ≺ b  — find x strictly inside every halfspace aᵢᵀx < bᵢ.

System II — certificate

λ ≠ 0, λ ≽ 0, Aᵀλ = 0, λᵀb ≤ 0 — a nonnegative combination of the rows that cancels, with λᵀb ≤ 0.

§6expert lens — design

D1The theorem

Nonempty disjoint convex C, D ⟹ ∃ a≠0, b with aᵀx ≤ b on C, aᵀx ≥ b on D — no closedness needed.

D2The construction

Closest points c, d; a = d − c; the plane bisects [c,d] perpendicularly (b at the midpoint). Not c+d, not through c or d.

D3Strict & the converse

Strict isn't always possible — but a point ∉ closed convex C always can ⟹ closed convex = ∩ of containing halfspaces. Converse needs one set open (C=D={0} fails otherwise).

D4Alternatives

Ax ≺ b infeasible ⟺ ∃ λ ≠ 0, λ ≽ 0, Aᵀλ = 0, λᵀb ≤ 0 — the Farkas / LP-duality prototype.

Main points for quick review

Theorem

Nonempty disjoint convex \(C, D\) ⟹ \(\exists a\neq0, b\) with \(a^{\mathsf T}x\le b\) on \(C\), \(a^{\mathsf T}x\ge b\) on \(D\). Non-strict needs no closedness/boundedness.

Construction

Closest points \(c\in C, d\in D\): \(a=d-c\); the plane perpendicularly bisects \([c,d]\) (\(b=(\|d\|^2-\|c\|^2)/2\) = \(a^{\mathsf T}\cdot\)midpoint).

Strict

Not always possible, but a point outside a closed convex \(C\) always can be ⟹ closed convex = ∩ of all containing halfspaces.

Alternatives

Converse needs one set open. \(Ax\prec b\) infeasible ⟺ \(\exists\lambda\neq0,\lambda\succeq0,A^{\mathsf T}\lambda=0,\lambda^{\mathsf T}b\le0\) — exactly one of the two systems is solvable.
