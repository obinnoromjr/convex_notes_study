Composition with an affine mapping

The second closure rule. Feed an affine map into the argument — g(x) = f(Ax + b) — and both convexity and concavity carry through untouched. One line to prove, and it underwrites half the "obviously convex" claims you ever make.

f — convex (or concave) on Rⁿ x ↦ Ax + b — the affine inner map g = f∘(Ax+b) — same curvature p. 79–80 · mastery: diagnose
§0where we are

This is the second convexity-preserving operation, right after nonnegative weighted sums (§3.2.1). It is short — a single rule — but it silently underwrites a huge number of "obvious" convexity claims. B&V state it without proof; the proof is a one-liner from the definition (3.1), so this sheet spends its effort on why the rule is exactly right and where the one classic slip hides.

+

What it buys you. Every one of these is "convex because it is a convex f precomposed with an affine map": the residual norm ‖Ax − b‖, log-barriers like −Σ log(bᵢ − aᵢᵀx), and quadratic residuals ‖Ax − b‖². Once you see the affine-composition rule, you stop re-checking (3.1) for each of them.

§1the rule
Suppose \(f:\mathbf R^{n}\to\mathbf R\), \(A\in\mathbf R^{n\times m}\), \(b\in\mathbf R^{n}\). Define \(g:\mathbf R^{m}\to\mathbf R\) by \[ g(x)=f(Ax+b),\qquad \operatorname{dom}g=\{x\mid Ax+b\in\operatorname{dom}f\}. \] Then if \(f\) is convex, so is \(g\); if \(f\) is concave, so is \(g\).B&V §3.2.2 · p. 79–80
  • where
    The affine map \(x\mapsto Ax+b\) acts on the argument — this is precomposition. It is not a postcomposition \(\alpha f(x)+\beta\), which scales the output.
  • both
    It preserves both curvatures. The affine inner map is "neutral": it does not flip convex to concave the way a general nonlinear inner map can.
§2the domain — the classic slip

The domain is not all of \(\mathbf R^{m}\): it is exactly the points \(x\) whose image \(Ax+b\) lands inside \(\operatorname{dom}f\).

\[ \operatorname{dom}g=\{x\mid Ax+b\in\operatorname{dom}f\} \] is the affine preimage of \(\operatorname{dom}f\), hence convex whenever \(\operatorname{dom}f\) is (§2.3.2). So "\(g\) convex" is well-posed.B&V §3.2.2 · p. 80
!

Getting the domain wrong is the classic slip. The statement "\(g\) is convex" needs a convex domain to even make sense. It is convex for free — an affine preimage of a convex set is convex — but only if you actually restrict to \(\{x\mid Ax+b\in\operatorname{dom}f\}\) rather than silently assuming \(g\) is defined everywhere.

§3proof from (3.1) — and why "affine" is exactly right

Take \(x,y\in\operatorname{dom}g\), \(\theta\in[0,1]\). Put \(u=Ax+b\) and \(v=Ay+b\), both in \(\operatorname{dom}f\). The whole proof turns on one identity.

g(x) = f(Ax+b) is convex when f isthe +b splits since θ+(1−θ)=1
\[ A\big(\theta x+(1-\theta)y\big)+b=\theta(Ax+b)+(1-\theta)(Ay+b)=\theta u+(1-\theta)v \]
the crucial step. The linear part handles \(A(\theta x+(1-\theta)y)\); the constant \(b\) splits only because \(\theta+(1-\theta)=1\). This is precisely why the inner map must be affine.
\[ g\big(\theta x+(1-\theta)y\big)=f\big(\theta u+(1-\theta)v\big) \]
apply the identity inside \(f\)
\[ \le\ \theta f(u)+(1-\theta)f(v)=\theta g(x)+(1-\theta)g(y). \]
(3.1) for convex \(f\) at \(u,v\), then unfold \(f(u)=g(x)\), \(f(v)=g(y)\). ∎
!

Concave \(f\): run the identical argument — every \(\le\) becomes \(\ge\). Nothing else changes, which is why the rule preserves concavity just as cleanly as convexity.

Why affine, not merely linear — and what breaks otherwise

The constant \(b\) survives the proof only because the weights sum to one. Drop affinity and the guarantee evaporates: a general nonlinear inner map can flip curvature. The book's own counterexample is the cleanest — take the convex \(f(u)=u\) and the non-affine inner map \(\varphi(x)=-x^2\); the composite is \(-x^2\), concave.

With \(f\) the identity, \(g=\varphi\). The affine map gives a convex (indeed affine) \(g\) — guaranteed, for any slope/shift. The −x² map breaks it: \(g=-x^2\) is concave. The x² map happens to stay convex — but that is luck, not the theorem. Affine is exactly the inner map that promises nothing breaks.

Smooth cross-check — the Hessian congruence

If \(f\) is twice differentiable, the chain rule gives the Hessian of \(g\) directly, and the sign is transparent.

\[ \nabla^{2}g(x)=A^{\mathsf T}\,\nabla^{2}f(Ax+b)\,A. \] When \(\nabla^{2}f\succeq 0\), this is a congruence \(A^{\mathsf T}MA\) of a PSD matrix, hence \(\succeq 0\): for any \(z\), \(z^{\mathsf T}A^{\mathsf T}MAz=(Az)^{\mathsf T}M(Az)\ge 0\).B&V §3.2.2 · p. 80
+

The one-dimensional shadow is \(g''(x)=a^{2}f''(ax+b)\): the factor \(a^{2}\ge 0\) can never flip the sign of \(f''\). That is the calculus echo of "the affine inner map is neutral."

§4expert lens — what to keep straight (diagnose)

D1The rule

f(Ax+b) is convex when f is convex and concave when f is concave — affine precomposition preserves both.

D2Argument, not output

It is the argument that is reparametrized: f(Ax+b), not αf(x)+β. And it is the inner map that must be affine.

D3Why affine, not linear

The constant b survives only because θ+(1−θ)=1. A non-affine inner map can destroy convexity: f(u)=u ∘ −x² gives −x², concave.

D4The domain

dom g = {x : Ax+b ∈ dom f}, the affine preimage of dom f — convex, but you must restrict to it. This is the classic slip.

D5Scope

This is affine precomposition only. Pointwise max / sup (§3.2.3) and general composition (§3.2.4) are separate rules with their own hypotheses.

Summary · 2026-07-20 · bv.3.2.2 · Composition with an affine mapping

Main points for quick review

The rule

\(g(x)=f(Ax+b)\) is convex if \(f\) convex, concave if \(f\) concave — affine precomposition preserves both.

The domain

\(\operatorname{dom}g=\{x\mid Ax+b\in\operatorname{dom}f\}\), the affine preimage of \(\operatorname{dom}f\) (convex).

The proof

Affinity sends \(\theta x+(1-\theta)y\mapsto\theta u+(1-\theta)v\) (the \(+b\) splits since \(\theta+(1-\theta)=1\)), then apply (3.1) to \(f\).

Smooth check

\(\nabla^{2}g=A^{\mathsf T}\nabla^{2}f(Ax+b)A\succeq0\) — a congruence of a PSD matrix.

Affine is exact

A non-affine inner map can break convexity: \(f(u)=u\ \circ\ -x^2=-x^2\).
