§0where we are

This is the last of §2.2's important examples, and the one that powers semidefinite programming. The objects are matrices, not vectors: the convex set lives in Sⁿ, the space of symmetric matrices. The star is Sⁿ₊, the cone of positive semidefinite matrices — convex by the same nonnegative-combination pattern as the norm cone (§2.2.3), now expressed through the quadratic form xᵀXx.

§1symmetric matrices Sⁿ — the ambient space
\(S^n=\{X\in\mathbf R^{n\times n}\mid X=X^{\mathsf T}\}\) — the symmetric \(n\times n\) matrices, a vector space of dimension \(n(n+1)/2\).B&V §2.2.5 · L1778

Why n(n+1)/2: a symmetric matrix is fixed by its diagonal (n entries) plus its strictly-upper triangle (n(n−1)/2); the lower triangle is forced by Xᵢⱼ = Xⱼᵢ. For n > 1, Sⁿ is a strict subspace of Rⁿˣⁿ (which has dimension n²); only at n = 1 do they coincide. This is the space the PSD cone sits inside.

§2the PSD / PD cones & convexity
\(S^n_+=\{X\in S^n\mid X\succeq0\}\) (positive semidefinite); \(\ S^n_{++}=\{X\in S^n\mid X\succ0\}\) (positive definite).B&V §2.2.5 · L1782
  • PSD
    X ≽ 0 ⟺ xᵀXx ≥ 0 for all x ⟺ all eigenvalues ≥ 0.
  • PD
    X ≻ 0 ⟺ xᵀXx > 0 for all x ≠ 0 ⟺ all eigenvalues > 0. PD ⟹ PSD, so Sⁿ₊₊ ⊆ Sⁿ₊.
⚠

Symbol overload. This ≽ is the matrix (Löwner) order — X ≽ 0 means PSD. It is a different relation from the componentwise ≼/≽ of §2.2.4 (where u ≼ v meant uᵢ ≤ vᵢ entrywise). Same symbol, different order.

Sⁿ₊ is a convex cone

Claim: \(\theta_1,\theta_2\ge0,\ A,B\in S^n_+ \Rightarrow \theta_1A+\theta_2B\in S^n_+\)straight from the definition
\[ x^{\mathsf T}(\theta_1 A+\theta_2 B)x = \theta_1\,(x^{\mathsf T}\!Ax) + \theta_2\,(x^{\mathsf T}\!Bx) \]
the quadratic form \(x^{\mathsf T}(\cdot)x\) is linear in the matrix — it splits the combination
\[ \ge 0 \qquad\text{for every } x \]
the crux: \(x^{\mathsf T}\!Ax\ge0,\ x^{\mathsf T}\!Bx\ge0\) (A,B PSD) and \(\theta_1,\theta_2\ge0\)
\[ \therefore\ \theta_1 A+\theta_2 B\succeq 0,\ \text{and it's symmetric.}\ \blacksquare \]
cone ← single scaling (\(\theta_2{=}0\)); convex ← \(\theta_1{+}\theta_2{=}1\)
!

One statement carries both properties. (Forward, not assessed: §2.3.1 re-proves convexity by writing Sⁿ₊ = ∩_{z≠0} {X | zᵀXz ≥ 0} — an intersection of infinitely many halfspaces.)

§3example 2.6 — diagnose a 2×2
\(X=\begin{bmatrix}x&y\\y&z\end{bmatrix}\in S^2_+ \iff x\ge0,\ z\ge0,\ xz\ge y^2\).B&V Example 2.6 · L1828

The 2×2 PSD test: nonnegative diagonal AND nonnegative determinant (det = xz − y² ≥ 0). All three are needed and independent — at x = 0, xz ≥ y² forces y = 0 but says nothing about z, so z ≥ 0 must be required separately. The boundary xz = y² is curved, so S²₊ is not a polyhedron — it's a rotated second-order cone, tying back to §2.2.3.

§4expert lens — diagnose

D1Know the ambient space

Sⁿ = symmetric n×n, dim n(n+1)/2 (n=2→3, n=3→6) — not Rⁿˣⁿ (dim n²).

D2Read the order correctly

X ≽ 0 ⟺ xᵀXx ≥ 0 ∀x. This ≽ is the Löwner order, not componentwise.

D3Justify the convex cone

xᵀ(θ₁A+θ₂B)x = θ₁xᵀAx + θ₂xᵀBx ≥ 0 for θ₁,θ₂ ≥ 0 (+ symmetric). Cone ← single scaling; convex ← θ₁+θ₂=1.

D4Test a 2×2

[[x,y],[y,z]] ≽ 0 ⟺ x≥0, z≥0, xz≥y² — nonneg diagonal + nonneg det, all three independent. All-nonnegative entries is neither necessary nor sufficient.

Main points for quick review

The space Sⁿ

Symmetric \(n\times n\) matrices, dim \(n(n+1)/2\). \(S^n_+\)= PSD \(\{X\succeq0\}\), \(S^n_{++}\)= PD \(\{X\succ0\}\).

The order

\(X\succeq0\iff x^{\mathsf T}Xx\ge0\ \forall x\) — the Löwner/matrix order, NOT the componentwise order of §2.2.4.

Convex cone

\(x^{\mathsf T}(\theta_1A+\theta_2B)x=\theta_1x^{\mathsf T}Ax+\theta_2x^{\mathsf T}Bx\ge0\) for \(\theta_i\ge0\). Quadratic form linear in the matrix; cone ← scaling, convex ← \(\theta_1{+}\theta_2{=}1\).

S²₊ test

\(\begin{bmatrix}x&y\\y&z\end{bmatrix}\succeq0\iff x\ge0,z\ge0,xz\ge y^2\) (nonneg diagonal + nonneg det, independent). Curved boundary ⟹ not a polyhedron (a rotated second-order cone).
