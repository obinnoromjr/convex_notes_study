§0where we are

§2.3.2 gave affine maps. §2.3.3 goes one step more general: maps that divide by a positive linear function of x. The building block is the perspective function P(z,t) = z/t; composing it with an affine map gives the linear-fractional functions f(x) = (Ax+b)/(cᵀx+d). Both still preserve convexity (image and inverse image) — and affine maps are the special case of a constant positive denominator.

§1the perspective function
\(P:\mathbf R^{n+1}\to\mathbf R^n\), \(\operatorname{dom}P=\mathbf R^n\times\mathbf R_{++}\) (last coordinate \(t>0\)), \(P(z,t)=z/t\).B&V §2.3.3 · L2019

Divide the first n coordinates by the last coordinate t, then drop t. Geometrically it's a pin-hole camera: the ray from the origin through (z,t) pierces the screen t = 1 at exactly (z/t, 1). A convex object, photographed, gives a convex image — that's the theorem.

Perspective maps segments to segmentsso convex ↦ convex
\[ P(\theta x+(1-\theta)y)=\frac{\theta\tilde x+(1-\theta)\tilde y}{\theta x_{n+1}+(1-\theta)y_{n+1}} \]
divide the blended numerator by the blended (positive) denominator
\[ =\ \mu\,P(x)+(1-\mu)\,P(y),\qquad \mu=\frac{\theta x_{n+1}}{\theta x_{n+1}+(1-\theta)y_{n+1}} \]
μ ∈ [0,1] and sweeps [0,1] monotonically as θ does → image is the segment [P(x), P(y)]
\[ \mu\ \neq\ \theta\ \ \text{in general.} \]
the subtlety: the division rescales the weight — the segment is preserved, not the midpoint
↔

Both directions. The same substitution gives the inverse image too: for convex C ⊆ Rⁿ, P⁻¹(C) = {(x,t) | x/t ∈ C, t > 0} is convex (with μ = θt/(θt + (1−θ)s) ∈ [0,1]). So perspective preserves convexity for image and inverse image.

§2linear-fractional = perspective ∘ affine
\(f(x)=\dfrac{Ax+b}{c^{\mathsf T}x+d}\), \(\operatorname{dom}f=\{x\mid c^{\mathsf T}x+d>0\}\).B&V §2.3.3 · L2109

A vector of linear functions over a common positive linear denominator. It's P ∘ g with g(x) = (Ax+b, cᵀx+d) affine — so it inherits convexity preservation from §2.3.2 (affine image) followed by §1 (perspective image). Two convexity-preservers composed.

⊇

Affine is the special case. If c = 0 and d > 0, the denominator is the constant d, dom f = Rⁿ, and f(x) = (Ax+b)/d is affine. So affine (and linear) functions are special linear-fractional functions — the new class strictly contains the old, not the reverse.

§3example 2.13 — conditional probabilities
\(f_{ij}=\operatorname{prob}(u=i\mid v=j)=\dfrac{p_{ij}}{\sum_{k=1}^n p_{kj}}\).B&V Example 2.13 · L2183

Fix a column j. The conditional distribution of u given v=j — (p₁ⱼ,…,pₙⱼ) / Σₖ pₖⱼ — is linear numerators over a common positive linear denominator Σₖ pₖⱼ = prob(v=j). That's exactly a linear-fractional map (a perspective/normalization). So a convex set of joint distributions maps to a convex set of conditional distributions.

§4expert lens — diagnose

D1Perspective

P(z,t) = z/t, domain t > 0 — divide by the last coordinate, drop it. Preserves convexity, image and inverse image.

D2The proof engine

P(θx+(1−θ)y) = μP(x)+(1−μ)P(y), μ = θxₙ₊₁/(θxₙ₊₁+(1−θ)yₙ₊₁) ∈ [0,1] monotone. μ ≠ θ — segment preserved, not midpoint.

D3Linear-fractional

f(x) = (Ax+b)/(cᵀx+d), domain cᵀx+d > 0 = perspective ∘ affine. Affine = special case c=0, d>0 (not the reverse).

D4Recognize-and-conclude

A ratio of linear-in-x over a positive linear denominator is linear-fractional ⟹ convex sets map to convex sets. Conditional probabilities (per column) is the model case.

Summary · 2026-06-30 · bv.2.3.3

Main points for quick review

Perspective

\(P(z,t)=z/t\), domain \(t>0\). Preserves convexity (image + inverse image) because it maps segments to segments.

Why (μ ≠ θ)

\(P(\theta x+(1-\theta)y)=\mu P(x)+(1-\mu)P(y)\), \(\mu=\frac{\theta x_{n+1}}{\theta x_{n+1}+(1-\theta)y_{n+1}}\), monotone in θ. The segment is preserved, not the midpoint.

Linear-fractional

\(f(x)=\frac{Ax+b}{c^{\mathsf T}x+d}\), domain \(c^{\mathsf T}x+d>0\) = perspective ∘ affine. Affine is the special case \(c=0,d>0\).

Recognize it

Ratio of linear-in-x over a positive linear denominator ⟹ linear-fractional ⟹ convex ↦ convex. Conditional probabilities (fix the conditioning value) is canonical.
