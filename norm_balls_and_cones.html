Norm balls
& norm cones

The §2.2.2 proof was coordinate-free — so it works for any norm. Cash that in: every norm ball is convex, and lifting the norm one dimension gives a convex cone whose Euclidean case is the famous second-order "ice-cream" cone.

§0where we are

§2.2.2 proved the Euclidean ball convex using only the triangle inequality and homogeneity — no coordinates anywhere. §2.2.3 cashes that in. The same proof works for any norm, so every norm ball is convex. Then it lifts the norm one dimension into the norm cone {(x,t) | ‖x‖ ≤ t} — a convex cone — whose Euclidean instance is the second-order cone. The skill here is to diagnose: justify convexity straight from the norm axioms, and recognize the SOC when you see it.

§1the norm ball — for any norm
For any norm \(\|\cdot\|\), the norm ball \(\{x \mid \|x-x_c\| \le r\}\) is convex — "from the general properties of norms."B&V §2.2.3 · L1579

This is the §2.2.2 proof verbatim with a general ‖·‖. That proof used only the triangle inequality and homogeneity — no coordinates — so it transfers unchanged. The Euclidean ball was never special. Different norms give different shapes — the ℓ₂ disk, the ℓ₁ diamond, the ℓ∞ box [−1,1]² — and every one of them is convex.

§2the norm cone — lift the norm one dimension
The norm cone is \(C=\{(x,t) \mid \|x\| \le t\} \subseteq \mathbf{R}^{n+1}\) — pairs \((x,t)\) where \(t\) bounds \(\|x\|\). It is a convex cone.B&V §2.2.3 · L1581

Mind the dimension: it lives in Rn+1 — the extra coordinate t, not Rⁿ. Both halves of "convex cone" fall straight out of the norm axioms:

Claim: \(C=\{(x,t)\mid\|x\|\le t\}\) is a convex conetwo axioms, two properties
\[ (x,t)\in C,\ \theta\ge0:\quad \|\theta x\| = \theta\|x\| \le \theta t \]
homogeneity gives \(\|\theta x\|=\theta\|x\|\); then scale \(\|x\|\le t\) by \(\theta\ge0\). So \((\theta x,\theta t)\in C\) → cone
\[ \|\theta x_1+(1-\theta)x_2\| \le \theta\|x_1\| + (1-\theta)\|x_2\| \]
triangle inequality + homogeneity split the combination
\[ \le \theta t_1 + (1-\theta)t_2. \qquad \therefore\ \text{combination}\in C.\ \blacksquare \]
bound each \(\|x_i\|\le t_i\) → convex
⌃

Homogeneity → the cone property; the triangle inequality → convexity. (Looking ahead, not assessed: the norm cone is the epigraph of the norm, so it's convex precisely because the norm is a convex function — that's §3.)

§3the second-order cone (example 2.3)
The second-order cone is the Euclidean norm cone: \(C=\{(x,t)\in\mathbf{R}^{n+1}\mid \|x\|_2 \le t\}\).B&V Example 2.3 · L1612

It has a clean quadratic-form representation:

\[ C = \Big\{ (x,t)\ \Big|\ \begin{bmatrix}x\\t\end{bmatrix}^{\!\mathsf T}\!\begin{bmatrix}I&0\\0&-1\end{bmatrix}\!\begin{bmatrix}x\\t\end{bmatrix}\le 0,\ t\ge 0\Big\}, \qquad \begin{bmatrix}x\\t\end{bmatrix}^{\!\mathsf T}\!\begin{bmatrix}I&0\\0&-1\end{bmatrix}\!\begin{bmatrix}x\\t\end{bmatrix}=x^{\mathsf T}x-t^2. \]

So xᵀx − t² ≤ 0 ⟺ ‖x‖₂² ≤ t², and (‖x‖₂² ≤ t² AND t ≥ 0) ⟺ ‖x‖₂ ≤ t.

!

The t ≥ 0 condition is essential. On its own, xᵀx ≤ t² gives |t| ≥ ‖x‖₂ — two nappes: an upper one (t ≥ ‖x‖₂) and a lower one (t ≤ −‖x‖₂). The t ≥ 0 clause selects the upper "ice-cream" cone. Other names: quadratic cone, Lorentz cone, ice-cream cone.

§4expert lens — diagnose

D1Any norm ball is convex

By the triangle inequality + homogeneity — coordinate-free, so it holds for every norm. Shapes differ (diamond, disk, box); all convex.

D2The norm cone is a convex cone

{(x,t) | ‖x‖ ≤ t} ⊆ Rn+1. Homogeneity → cone; triangle inequality → convex. Always remember it lives in n+1 dimensions.

D3Recognize the SOC

Second-order cone = the Euclidean norm cone. Rep xᵀx ≤ t² with t ≥ 0 (the t≥0 picks the upper nappe).

D4Membership

(x,t) is in the norm cone iff ‖x‖ ≤ t — on the boundary when ‖x‖ = t.

Main points for quick review

Norm ball

Every \(\{\|x-x_c\|\le r\}\) is convex (any norm) — the §2.2.2 proof is coordinate-free. ℓ₁ diamond, ℓ∞ box, ℓ₂ disk: all convex.

Norm cone

\(\{(x,t)\mid\|x\|\le t\}\subseteq\mathbf{R}^{n+1}\) is a convex cone: homogeneity → cone, triangle inequality → convex. Note the \(n{+}1\) dims.

Second-order cone

= Euclidean norm cone \(\{\|x\|_2\le t\}\); quadratic rep \(x^{\mathsf T}x\le t^2\) with \(t\ge0\) (picks the upper nappe). aka quadratic / Lorentz / ice-cream cone.
