§0where we are

§2.5.1 separated two disjoint convex sets. §2.5.2 is the boundary counterpart: a hyperplane that touches one convex set at a boundary point and keeps the whole set on one side. It falls straight out of the separating theorem — and it's the geometric seed of subgradients and the dual description of convex sets.

§1definition & the theorem

Fix a boundary point x₀ ∈ bd C = cl C \ int C — a point in the closure of C that is not interior.

Let \(x_0\in\operatorname{bd}C\). If \(a\neq0\) satisfies \(a^{\mathsf T}x\le a^{\mathsf T}x_0\) for all \(x\in C\), then \(\{x\mid a^{\mathsf T}x=a^{\mathsf T}x_0\}\) is a supporting hyperplane to \(C\) at \(x_0\).B&V §2.5.2 · L2589

The hyperplane passes through x₀ (offset b = aᵀx₀), is tangent to C at x₀, and the halfspace {aᵀx ≤ aᵀx₀} contains all of C — the whole set on one side, touching at x₀. (The ≤ side is a convention; using −a flips it.)

For any nonempty convex \(C\) and any \(x_0\in\operatorname{bd}C\), there exists a supporting hyperplane to \(C\) at \(x_0\).B&V §2.5.2 · L2616

§2only boundary points have one

The theorem is about boundary points. An interior point never has a supporting hyperplane: any hyperplane through an interior point cuts C on both sides, so it can't keep the whole set on one side.

§3the dual description & the converse
Partial converse: if a set is closed, has nonempty interior, and has a supporting hyperplane at every boundary point, then it is convex.B&V §2.5.2 · L2624

This is the boundary-side companion to §2.5.1's fact that a closed convex set is the intersection of its containing halfspaces. Sample supporting halfspaces all around the boundary and their intersection rebuilds the set — if it's convex. A non-convex dent has boundary points with no supporting hyperplane, and the intersection overshoots to the convex hull.

§4expert lens — diagnose

D1The definition

At x₀ ∈ bd C: a ≠ 0, aᵀx ≤ aᵀx₀ ∀ x ∈ C; the hyperplane passes through x₀, is tangent, and C sits on the ≤ side.

D2The theorem

Every boundary point of a nonempty convex set has one — no closedness/boundedness needed. Interior points have none.

D3Proof idea & uniqueness

Separate {x₀} from int C, extend by closure. Unique at smooth boundary points; a whole fan at corners.

D4Partial converse

Closed + nonempty interior + a supporting hyperplane at every boundary point ⟹ convex.

Main points for quick review

Definition

At \(x_0\in\operatorname{bd}C\): \(a\neq0\) with \(a^{\mathsf T}x\le a^{\mathsf T}x_0\ \forall x\in C\); the hyperplane through \(x_0\) is tangent, \(C\) in the halfspace \(\{a^{\mathsf T}x\le a^{\mathsf T}x_0\}\).

Theorem

Every nonempty convex set has a supporting hyperplane at every boundary point (no closedness/boundedness). Interior points don't.

Proof

If \(\operatorname{int}C\neq\emptyset\), separate \(\{x_0\}\) from \(\operatorname{int}C\) and extend by closure; if \(\operatorname{int}C=\emptyset\), \(C\) lies in a lower-dimensional affine set.

Converse & uniqueness

Closed + nonempty interior + support at every boundary point ⟹ convex. Unique at smooth points, a fan at corners.
