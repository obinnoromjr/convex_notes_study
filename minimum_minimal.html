Minimum &
minimal elements

A generalized inequality isn't a total order — two points can be incomparable. Once that happens, the single scalar idea of "the smallest" splits into two: a minimum that beats everything, and minimal ones that just can't be beaten.

minimum: S ⊆ x+K minimal: (x−K)∩S={x} incomparable pairs p. 45–46 · mastery: diagnose
§0why "smallest" splits in two

§2.4.1 built ≼_K. It shares much of the algebra of ≤, but it's missing one property: ≤ on R is a total order — any two numbers are comparable. A general ≼_K is not: two points can be incomparable (neither x ≼_K y nor y ≼_K x). Once comparability can fail, the single scalar idea of "the smallest element" splits into two distinct concepts — minimum and minimal — that coincide on R but differ in general.

§1two definitions, two set-tests
  • minimum
    x is the minimum of S if x ≼_K y for every y ∈ S — it dominates from below. Strong; unique if it exists; often doesn't. Test: S ⊆ x + K.
  • minimal
    x is a minimal element if y ∈ S, y ≼_K x ⟹ y = x — nothing in S is below it. Weak, local; a set can have many. Test: (x − K) ∩ S = {x}.
≠

The contrast in one line: minimum = x is ≼_K everything (dominates all) → unique. Minimal = nothing is ≼_K x (nothing beats it) → possibly many. The gap between them is exactly the incomparable pairs. Minimum uses x + K (everything above x); minimal uses x − K (nothing below x in S except x).

§2example 2.18 — ellipsoids, no minimum
For \(A\in S^n_{++}\), \(\mathcal E_A=\{x\mid x^{\mathsf T}A^{-1}x\le1\}\). Key fact: \(A\preceq B\) (Löwner) \(\iff \mathcal E_A\subseteq\mathcal E_B\) — the matrix order is ellipsoid containment.B&V Example 2.18 · L2340

Let S be the ellipsoids (centered at 0) containing given points v₁,…,vₖ. There's no minimum: for any containing ellipsoid there's another, incomparable one — shrink one axis, grow another, still contain the points. But there are minimal ones: containing ellipsoids with no smaller containing ellipsoid inside them (inclusion-minimal, not a volume objective).

For axis-aligned A = diag(a², b²), containment E_A ⊆ E_B ⟺ a ≤ a', b ≤ b' — the Löwner order becomes the componentwise order on (a, b). So this is Lab 1 in disguise: the minimal ellipsoids are the Pareto frontier of the feasible (a, b) region.

§3on R the split disappears

For K = R₊, minimum and minimal coincide and agree with the ordinary minimum. The split is a genuinely generalized-inequality phenomenon — it appears only once the order stops being total. Watch it happen: flatten the second coordinate and the plane collapses to a line.

§4expert lens — diagnose

D1Not a total order

≼_K generally has incomparable pairs — that's why "smallest" splits into minimum vs minimal.

D2Minimum

x ≼_K y for every y → unique. Test S ⊆ x + K.

D3Minimal

Nothing in S is ≼_K x except x → possibly many. Test (x − K) ∩ S = {x}.

D4Relations & move

Minimum ⟹ minimal, not conversely; they coincide only on a total order (K = R₊). Minimal points are the undominated ones — the Pareto frontier.

Main points for quick review

Why it splits

\(\preceq_K\) is not a total order — two points can be incomparable — so "smallest" splits into minimum vs minimal.

Minimum

\(x\preceq_K y\) for every \(y\in S\) (dominates all) — unique, often doesn't exist; \(\iff S\subseteq x+K\).

Minimal

Nothing in S is \(\preceq_K x\) except x (undominated) — possibly many; \(\iff (x-K)\cap S=\{x\}\).

Relations

Minimum ⟹ minimal, not conversely; coincide on a total order (\(K=\mathbf R_+\)). Ex. 2.18: containing ellipsoids have no minimum, but minimal = inclusion-minimal (the Pareto frontier).
