Sublevel
sets

The bridge back to convex sets: if \(f\) is convex, every set \(\{x\mid f(x)\le\alpha\}\) is convex. This is the workhorse for proving a set convex — recognize it as a sublevel set. The one subtlety to burn in: the implication runs one way only.

Cα = {f ≤ α} convex ⟹ every Cα convex converse fails: −eˣ p. 74–75 · mastery: diagnose
§0where we are

§3.1.1–3.1.5 characterized convex functions from the inside — the definition, the gradient and Hessian tests, the catalogue of examples. §3.1.6 connects them back to convex sets: the sublevel sets of a convex function are convex. This is a workhorse for proving a set is convex — you recognize it as a sublevel set of a convex function (or a superlevel set of a concave one). The one subtlety to internalize is that the implication runs one direction only: convex functions have convex sublevel sets, but a function with convex sublevel sets need not be convex.

§1sublevel sets
The \(\alpha\)-sublevel set of \(f:\mathbb{R}^n\to\mathbb{R}\) is \[ C_\alpha=\{x\in\operatorname{dom}f\mid f(x)\le\alpha\}. \]B&V §3.1.6 · L3782–3783

One set per level \(\alpha\): the points where \(f\) is at most \(\alpha\). As \(\alpha\) rises the set grows, sweeping out \(\operatorname{dom}f\) from the low ground of \(f\) upward. The whole section is about the shape of these sets.

§2convex ⟹ all sublevel sets convex
If \(f\) is convex, then \(C_\alpha\) is convex for every value of \(\alpha\).B&V §3.1.6 · L3784–3786

The proof is immediate from the defining inequality (3.1). The point that makes it work: a convex combination of two numbers both \(\le\alpha\) is again \(\le\alpha\), because \(\theta\alpha+(1-\theta)\alpha=\alpha\). The level is preserved.

Claim — \(f\) convex \(\Rightarrow C_\alpha\) convexthe level survives
\[ x,y\in C_\alpha\ \Longrightarrow\ f(x)\le\alpha,\quad f(y)\le\alpha \]
setup take any two members of the sublevel set; each has value at most \(\alpha\).
\[ f(\theta x+(1-\theta)y)\ \le\ \theta f(x)+(1-\theta)f(y) \]
convexity of \(f\), inequality (3.1), for \(0\le\theta\le 1\).
\[ \le\ \theta\alpha+(1-\theta)\alpha\ =\ \alpha \]
crux bound each term by \(\alpha\); the weights sum to \(1\), so the level \(\alpha\) comes through unchanged.
\[ \therefore\ \theta x+(1-\theta)y\in C_\alpha \]
the whole segment lies in \(C_\alpha\): it is convex. \(\blacksquare\)
!

This is the workhorse. To certify that a set is convex, write it as \(\{x\mid f(x)\le\alpha\}\) for a convex \(f\). The Euclidean ball \(\{x\mid\lVert x\rVert_2\le 1\}\), for instance, is the \(1\)-sublevel set of the (convex) norm \(\lVert\cdot\rVert_2\) — so it is convex, no direct segment argument needed.

§3the converse fails

The implication does not reverse. A function can have all its sublevel sets convex and still not be convex.

Counterexample: \(f(x)=-e^{x}\) on \(\mathbb{R}\) is strictly concave (\(f''=-e^{x}<0\)), hence not convex. Yet since \(-e^{x}<0\) always, every sublevel set is \(\mathbb{R}\) or a half-line — both convex.B&V §3.1.6 · L3787–3789
  • α ≥ 0
    \(-e^{x}\le\alpha\) holds for all \(x\) (the graph sits entirely below \(0\)) \(\Rightarrow C_\alpha=\mathbb{R}\).
  • α < 0
    \(-e^{x}\le\alpha\iff e^{x}\ge-\alpha\iff x\ge\log(-\alpha)\ \Rightarrow C_\alpha=[\log(-\alpha),\infty)\), a half-line.
!

"All sublevel sets convex" is strictly weaker than "convex". Every \(C_\alpha\) here is \(\mathbb{R}\) or a half-line — impeccably convex — yet \(f\) is not. (The property "all sublevel sets convex" defines the quasiconvex functions, developed later in the chapter; convexity is a strictly stronger condition.)

§4concave ⟹ superlevel sets convex
By the mirror argument, if \(f\) is concave, its \(\alpha\)-superlevel set \[ \{x\in\operatorname{dom}f\mid f(x)\ge\alpha\} \] is convex.B&V §3.1.6 · L3790–3796

The proof is the same with the inequalities flipped: for concave \(f\), \(f(\theta x+(1-\theta)y)\ge\theta f(x)+(1-\theta)f(y)\ge\theta\alpha+(1-\theta)\alpha=\alpha\). This gives the second half of the standard convexity-certification toolkit.

  • convex f
    Express the set as \(\{x\mid f(x)\le\alpha\}\) — a sublevel set of a convex function.
  • concave f
    Or express it as \(\{x\mid f(x)\ge\alpha\}\) — a superlevel set of a concave function.
i

The move to practise: when asked to show a set is convex, hunt for the function whose sublevel (or superlevel) set it is. Half-spaces, balls, ellipsoids, and the AM–GM set below all fall out of this one idea.

§5example 3.3 — the AM–GM set is a convex cone

For \(x\in\mathbb{R}^n_{+}\) define the geometric and arithmetic means

\[ G(x)=\Big(\textstyle\prod_{i=1}^{n}x_i\Big)^{1/n}\ \ (\text{with }0^{1/n}=0),\qquad A(x)=\frac1n\sum_{i=1}^{n}x_i, \] and AM–GM gives \(G(x)\le A(x)\). For \(0\le\alpha\le 1\), the set \[ \{\,x\in\mathbb{R}^n_{+}\mid G(x)\ge\alpha\,A(x)\,\} \] — the vectors whose geometric mean is at least a fraction \(\alpha\) of their arithmetic mean — is convex.B&V §3.1.6 · Example 3.3 · L3798–3814
  • why convex
    It is the \(0\)-superlevel set \(\{h\ge 0\}\) of \(h=G-\alpha A\). Here \(G\) is concave on \(\mathbb{R}^n_{+}\) (B&V's \(0^{1/n}=0\) convention) and \(-\alpha A\) is linear, so \(h\) is concave \(\Rightarrow\) its superlevel set is convex, by §4.
  • why a cone
    \(G\) and \(A\) are degree-1 positively homogeneous, so \(h(tx)=t\,h(x)\) for \(t\ge 0\). Then \(h(x)\ge 0\Rightarrow h(tx)\ge 0\): the set is a cone.
!

Convex + cone \(\Rightarrow\) convex cone. The superlevel-set argument gives convexity; the homogeneity of \(G-\alpha A\) gives the cone property. Together the AM–GM set is a convex cone.

§6worked examples
Example 1 — the sublevel sets of \(x^2\)intervals

Take \(f(x)=x^2\) on \(\mathbb{R}\) (convex). Its \(\alpha\)-sublevel set is

\[ C_\alpha=\{x\mid x^2\le\alpha\}=\begin{cases}\varnothing,&\alpha<0,\\[2pt]\{0\},&\alpha=0,\\[2pt][-\sqrt\alpha,\ \sqrt\alpha],&\alpha>0.\end{cases} \]

Every case is convex — an empty set, a point, or a closed interval. Note that the empty set and singletons are convex too, so the "for every \(\alpha\)" clause needs no exceptions.

Example 2 — the \(-e^{x}\) converse, computedconcave, still convex Cα

\(f(x)=-e^{x}\), \(f''(x)=-e^{x}<0\): strictly concave, so not convex. Its sublevel sets:

\[ \alpha\ge 0:\quad -e^{x}\le\alpha\ \text{for all }x\ \Rightarrow\ C_\alpha=\mathbb{R}. \]

\[ \alpha<0:\quad -e^{x}\le\alpha\iff e^{x}\ge-\alpha\iff x\ge\log(-\alpha)\ \Rightarrow\ C_\alpha=[\log(-\alpha),\infty). \]

Both are convex for every \(\alpha\). So a function can have all-convex sublevel sets without being convex: the converse of §2 is false, and "convex sublevel sets" (quasiconvexity) is the strictly weaker property.

Example 3 — the unit ball as a sublevel setthe certification move

The Euclidean unit ball \(B=\{x\mid\lVert x\rVert_2\le 1\}\) is exactly the \(1\)-sublevel set of \(f(x)=\lVert x\rVert_2\). A norm is convex (triangle inequality + homogeneity), so by §2 the ball is convex — with no direct \(\theta x+(1-\theta)y\) computation. The same argument works for any norm ball \(\{x\mid\lVert x\rVert\le r\}\), and for ellipsoids via \(f(x)=\lVert A^{-1}(x-x_c)\rVert_2\).

Example 4 — the AM–GM set with \(n=2,\ \alpha=1\)the wedge is a single ray

For \(n=2\): \(G=\sqrt{x_1x_2}\), \(A=(x_1+x_2)/2\). The condition \(G\ge\alpha A\) with \(\alpha=1\) is

\[ \sqrt{x_1x_2}\ \ge\ \tfrac12(x_1+x_2)\iff 2\sqrt{x_1x_2}\ge x_1+x_2\iff 0\ge(\sqrt{x_1}-\sqrt{x_2})^2. \]

A square is \(\ge 0\), so this forces \(\sqrt{x_1}=\sqrt{x_2}\), i.e. \(x_1=x_2\): the set is the single ray \(\{(t,t)\mid t\ge 0\}\) — a (degenerate) convex cone, exactly the equality case of AM–GM. For \(\alpha<1\) the equality relaxes to a wedge of rays around this diagonal, widening to the whole quadrant \(\mathbb{R}^2_+\) as \(\alpha\to 0\).

§7expert lens — diagnose

D1Definition

\(C_\alpha=\{x\in\operatorname{dom}f\mid f(x)\le\alpha\}\) — one sublevel set per level \(\alpha\).

D2Convex ⟹ every Cα convex

The level is preserved by convex combinations: \(\theta\alpha+(1-\theta)\alpha=\alpha\). Mirror: \(f\) concave ⟹ every superlevel set \(\{f\ge\alpha\}\) convex.

D3One-directional

All sublevel sets convex does not imply \(f\) convex — witness \(-e^{x}\), concave with sublevel sets \(\mathbb{R}\) or half-lines. That weaker property is quasiconvexity.

D4Use it to prove sets convex

Write the set as \(\{f\le\alpha\}\) (\(f\) convex) or \(\{f\ge\alpha\}\) (\(f\) concave). Balls, ellipsoids, half-spaces all follow.

D5Example 3.3

\(\{G\ge\alpha A\}\) is convex because \(G-\alpha A\) is concave, and a convex cone because \(G-\alpha A\) is degree-1 homogeneous.

Main points for quick review

Definition

\(C_\alpha=\{x\in\operatorname{dom}f\mid f(x)\le\alpha\}\). \(f\) convex \(\Rightarrow C_\alpha\) convex for every \(\alpha\) (proof: \(\theta\alpha+(1-\theta)\alpha=\alpha\)).

Converse fails

\(-e^{x}\) is strictly concave yet all its sublevel sets (\(\mathbb{R}\) or half-lines \([\log(-\alpha),\infty)\)) are convex.

Concave mirror

\(f\) concave \(\Rightarrow\) superlevel sets \(\{f\ge\alpha\}\) convex.

Workhorse

Prove a set convex by writing it as \(\{f\le\alpha\}\) (convex \(f\)) or \(\{f\ge\alpha\}\) (concave \(f\)) — e.g. the ball \(\{\lVert x\rVert_2\le 1\}\).

Example 3.3

\(\{G\ge\alpha A\}\) on \(\mathbb{R}^n_+\) is a convex cone — \(0\)-superlevel set of the concave \(G-\alpha A\), plus degree-1 homogeneity.
