Affine functions

The workhorse of the calculus. The rule is two-sided — both the image and the inverse image of a convex set stay convex — and it certifies the polyhedron, the ellipsoid, the SOC region, and the whole SDP feasible set in one stroke.

image f(S) inverse image f⁻¹(T) stays convex p. 37–39 · mastery: design
§0where we are

§2.3.1 gave the first convexity-preserving operation, intersection. §2.3.2 gives the workhorse: affine functions. The rule is two-sided — both the image and the inverse image of a convex set are convex — and many concrete convex sets are certified this way: the polyhedron, the ellipsoid, the SOC region, and the SDP/LMI feasible set are all affine images or inverse images of a simpler convex set.

§1the two rules — one identity
\(f:\mathbf R^n\to\mathbf R^m\) is affine if \(f(x)=Ax+b\). Rule 1 (image): \(S\) convex ⟹ \(f(S)\) convex. Rule 2 (inverse image): \(T\) convex ⟹ \(f^{-1}(T)=\{x\mid f(x)\in T\}\) convex.B&V §2.3.2 · L1897, L1943
The one identity behind bothaffine preserves convex combinations
\[ f(\theta u+(1-\theta)v)=A(\theta u+(1-\theta)v)+b=\theta(Au+b)+(1-\theta)(Av+b) \]
the constant \(b\) recombines because \(\theta+(1-\theta)=1\)
\[ =\ \theta f(u)+(1-\theta)f(v) \]
a convex combination maps to a convex combination
\[ \text{image: }\theta f(u){+}(1{-}\theta)f(v)=f(\underbrace{\theta u{+}(1{-}\theta)v}_{\in S})\ \ \big|\ \ \text{inv: }f(\underbrace{\theta u{+}(1{-}\theta)v}_{?})=\underbrace{\theta f(u){+}(1{-}\theta)f(v)}_{\in T} \]
image uses S convex (domain); inverse image uses T convex (codomain)
⊞

The Cartesian-product move. To handle two sets at once, stack into a product S₁ × S₂ (still convex), then apply a linear map. Set sum S₁ + S₂ is the image of S₁ × S₂ under f(x₁,x₂) = x₁ + x₂; projection drops a coordinate; partial sum interpolates (m=0 → intersection, n=0 → addition).

§2the four examples — certify by a map

The design skill is to spot the known convex set + the affine map that produces yours. Three inverse images of a cone, one image of a ball:

Example 2.9 · inverse image

Polyhedron ← nonnegative orthant

{x | Ax ≼ b, Cx = d} = f⁻¹(Rᵐ₊ × {0}) with f(x) = (b − Ax, d − Cx). (Ax ≼ b ⟺ b − Ax ≽ 0; Cx = d ⟺ d − Cx = 0.)

Example 2.10 · inverse image

LMI solution set ← PSD cone

A linear matrix inequality A(x) = x₁A₁ + ··· + xₙAₙ ≼ B — the matrix analogue of aᵀx ≤ b. Its solution set is f⁻¹(Sᵐ₊), f(x) = B − A(x). This is the SDP feasibility constraint — the bridge from §2.2.5 to semidefinite programming.

Example 2.11 · inverse image

Hyperbolic cone ← second-order cone

{x | xᵀPx ≤ (cᵀx)², cᵀx ≥ 0} (P ≽ 0) is f⁻¹(SOC) under f(x) = (P^½x, cᵀx): then zᵀz = xᵀPx, t = cᵀx.

Example 2.12 · image

Ellipsoid = image of the unit ball

E = {x | (x−xc)ᵀP⁻¹(x−xc) ≤ 1} is the image of {u | ‖u‖₂ ≤ 1} under f(u) = P^½u + xc. Closes the §2.2.2 loop: image form and inequality form are the same set.

§3image — example 2.12

The ellipsoid is the image of the unit ball under f(u) = P^½u + xc. The map P^½ stretches the ball by √λᵢ along P's eigenvectors; xc shifts it. Watch a point on the ball ride to its image on the ellipsoid.

§4inverse image — example 2.11

The hyperbolic cone is the inverse image of the second-order cone. The map f(x) = (P^½x, cᵀx) sends each x to a pair (z, t); pulling back the SOC {‖z‖₂ ≤ t} gives {x | xᵀPx ≤ (cᵀx)², cᵀx ≥ 0}. The convexity is free — it's f⁻¹ of a convex cone.

§5expert lens — design

D1Both directions

Affine f(x)=Ax+b (linear PLUS constant). Image f(S) and inverse image f⁻¹(T) of convex sets are convex — affine maps respect convex combinations.

D2The corollaries

Scaling, translation, projection, set sum (Cartesian product + linear map), partial sum — all special cases.

D3Inverse image of a cone

Polyhedron ← orthant; LMI ← PSD cone; hyperbolic cone ← SOC. Image of a set: ellipsoid = image of the unit ball.

D4The master pattern

A constraint g(x) ∈ K with g affine and K convex is convex. Orthant → LP, SOC → SOCP, PSD cone → SDP.

Main points for quick review

The two rules

Affine \(f(x)=Ax+b\). Both image \(f(S)\) and inverse image \(f^{-1}(T)\) of convex sets are convex — affine maps preserve convex combinations (the constant \(b\) recombines, weights sum to 1).

Corollaries

Scaling \(\alpha S\), translation \(S+a\), projection, set sum \(S_1+S_2\) (product imaged under \((x_1,x_2)\mapsto x_1+x_2\)), partial sum.

Certify by a map

Inverse image of a cone: polyhedron ← orthant; LMI ← \(S^m_+\); hyperbolic cone ← SOC. Image: ellipsoid = image of unit ball, \(f(u)=P^{1/2}u+x_c\).

The pattern

\(g(x)\in K\) with \(g\) affine, \(K\) convex ⟹ convex. Orthant→LP, SOC→SOCP, PSD cone→SDP.
